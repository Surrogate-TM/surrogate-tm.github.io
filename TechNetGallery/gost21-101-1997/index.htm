Набор элементов ГОСТ 21.101-97

Архив моих работ с TechNetGallery

Предлагаю вашему вниманию набор инструментов для создания документа MS Visio, оформленного в соответствии с ГОСТ 21.101-97. Всё что нужно для создания документа, находится в одном наборе элементов (vss).

Изначально набор был доступен по адресу


Состав

В состав данного набора относятся фигуры имитирующие ряд форм из ГОСТ 21.101-97 (Формы 1,2,3,5,6,7,14,15 и 16), спецификацию по ГОСТ 21.110-95 и две фигуры имеющие утилитарное значение.

Причем некоторые из фигур сочетают в себе несколько форм.

Для заполнения общих полей документа нужно воспользоваться фигурой «Заполнить форму». Перетаскиваем данную фигуру из набора элементов

После вытаскивания фигуры на лист, активируется окно ввода данных.

Вам необходимо заполнить поля на форме и нажать кнопку OK.

Для повторного вызова данного окна дважды щелкните мышью по фигуре «Заполнить форму» на листе.

Навигация между пунктами меню: вниз – Tab, вверх - Shift+Tab.

В некоторых полях (Разработал, Проверил, ГИП, Н.котроль, Утвердил, Форма собственности организации и Стадия) существует возможность пополнения списка – таким образом вы можете создать в данном шаблоне список ваших сотрудников, форм собственности и стадий разработки документации. В дальнейшем вы сможете выбирать вариант из выпадающего списка, а не вводить его руками.

В поле «Тип организации» новые значения вводятся в формате: «Аббревиатура(черта)Расшифровка аббревиатуры».

Например «ГП-Государственное предприятие». Аналогичным образом заполняются варианты списка для поля «Стадия». В настоящее время реализован механизм подгонки шрифтов в этих полях по ширине. Ограничение длина поля не более 12 символов. По умолчанию подставляется дата cоздания документа. При необходимости изменения даты производится выбор значения из календаря.

Все вышеперечисленные данные будут заполняться на всех страницах документа автоматически !!!

Мастер «Титул_Обл»

Данный мастер реализован в соответствии с формами №14 и №15 из ГОСТ 21.101-97. По умолчанию первая страница в документе считается Обложкой, вторая Титулом.

На титульном листе можно добавить подписи – для этого щелкните правой кнопкой мыши на рамке. В самом верху появится пункт меню «Добавить подписи». Или убрать подписи аналогично.

Мастер «Рамка»

При вытаскивании данного мастера появляется форма с полями для заполнения основной надписи на текущем листе документа.

В поле «Форма основной надписи» указываем необходимую форму основной надписи в соответствии с ГОСТ 21.101-97 на текущем листе.

В данном мастере реализованы формы №3, №5 и №6. В поле «Наименование листа» указываем условное обозначение, которое будет использоваться и наименование текущего листа.

Данный список является пополняемым и/или редактируемым.

Для повторного вызова окна ввода данных этой фигуры необходимо ее выделить. Далее щелкнуть правой кнопкой мыши и выбрать самый верхний пункт выпавшего контекстного меню - «Окно свойств».

На отдельных листах можно устанавливать фамилии в полях (Разработал, Проверил, ГИП, Н.котроль, Утвердил) отличные от надписей на остальных страницах документа.Для этого нужно навести мышь на интересующее поле, щелкнуть по правой кнопкой мыши и выбрать пункт «Изменить». В появившемся окне ввести новую фамилию.

Нумерация страниц автоматическая, если есть необходимость изменить номер страницы – щелкните правой кнопкой мыши на рамке. В самом верху появится пункт меню «Номер страницы».

Мастер «Спецификация»

Этот мастер я задумал по причине того, что мне не нравилось как выглядит спецификация вставленная в Visio из Excel.

Чтоб воспользоваться этим инструментом, нужно предварительно перенести свою спецификацию в Excel в файл спецификация.xls из архива. В этом файле подобраны высоты строк для отображения на первой странице 25 строк (случай на странице применяется Форма №3), на всех последующих страницах должно отражаться по 30 строк (случай на странице применяется Форма №6). Если вы применяете на странице Форму №5 ,то на странице должно умещаться 27 строк.

Ширина столбцов также приближена к тому как это будет выглядеть на листе Visio.

Есть два способа заполнения таблицы:

1. Предварительно скопировать нужное вам количество строк в Excel-евской таблице и вытащить мастер на лист. При этом появится вопрос окно диалога Paste special: из представленных вариантов необходимо выбрать Unformatted text или Unformatted Unicode text и нажать Ok.

┌─────────────────────────────────────────────────┐

│При копировании нужно ОБЯЗАТЕЛЬНО скопировать первый пустой столбец А │

└─────────────────────────────────────────────────┘

2. Редактировать текст можно уже во вставленном мастере. Для этого можно нажать F2 или два раза щелкнуть мышью на таблице спецификации.

Заполняем содержимое "ячейки" таблицы, для перехода к "ячейке" в следующем столбце нажать клавишу Tab на клавиатуре. Для перевода курсора для ввода текста на следующей строке нажимаем кнопку Enter.

Для навигации по тексту пользуемся комбинацией Ctrl и стрелок навигации ←, ↑, ↓ и →.

Ctrl+→ -Вправо

Ctrl+← -Влево

Ctrl+↑ -К началу предыдущей строки

Ctrl+↓ -К началу следущей строки

В данном мастере нет никаких хитростей. Просто подобрано положение Tabs`ов и отступы между строками.

Фигуры « 21.101-97-F1 » , « 21.101-97-F2», « 21.101-97-F7» и « 21.101-97-F16» работают аналогично фигуре « Спецификация».

Мастер «Save as ..»

Данный мастер используется для более удобного присвоения файлам названий, с целью минимизировать ручной ввод данных. вытащить трафарет на лист, убедиться что надпись в поле сответствует желаемому названию файла. Нажать Ctrl+C, далее щелкнуть правой кнопкой мыши «Сохранить». в выпавшем окне нажать Ctrl+V.По умолчанию название файла будет состоять из Шифра проекта и текущей даты.

Заключение

В каждой из фигур данного набора элементов есть ссылка на ресурс, в котором вы можете найти ссылку на скачивание (на тот случай если Вы забудете где его скачали), видео-описания и т.п. . Гиперсылку вышеуказанную страницу можно найти щелкнув мышью при зажатой клавише Ctrl на любую из фигур вытащенных на лист. Или щелкнув на фигуре правой кнопкой мыши, в выбрать в выпавшем контекстном меню пункт «Описание трафарета Visio_document_2020»
