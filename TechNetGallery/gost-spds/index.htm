MS Visio. Набор элементов ГОСТ Р 21.1101-2013. Новый механизм ввода данных

Данный архив предназначен, для создания многостраничных документов в оформленных в соответствии с требованиям Системы проектной документации для строительства (СПДС). Может быть полезен широкому спектру пользователей: студентам, инженерам, техническим писателям. Архив содержит ряд фигур содержащихся в наборах ГОСТ Р 21.1101-2013, ГОСТ 21.501-2011 и ГОСТ 21.608-2014. Обновление от 30.06.2020 - появилась возможность изменять используемый шрифт и добавлять наклонное начертание. Форма 6 Описание ГОСТ Р 21.1101-2013 В состав данного набора относятся фигуры имитирующие ряд форм из ГОСТ Р 21.1101-2013 (Формы 1,2,3,5,6,7,12,13 и 14), спецификацию по ГОСТ 21.110-2013 (Форма 1). Мастер «Титул_обложка» Внешний вид мастера (Приложение Н. Форма 12 или 13) зависит от того на каком листе по порядку документа он вставлен. На первом листе вид будет соответствовать обложке (форма 12), на втором титульному листу (форма 13). На верхней границе фигуры есть желтая точка с помощью, которой можно можно регулировать ширину поля для размещения логотипа. На левой стороне фигуры расположен ряд аналогичных точек, для регулировки местоположения текстовых полей на титульном листе/обложке. На листе Обложка, возможно изменить видимость надписей для согласующих лиц. Для этого нужно выделить фигуру, щелкнуть правой кнопкой мыши и выбрать пункт меню «Показать подписи». Мастер «Рамка» Изменить внешний вид данной фигуры (Приложение Ж, формы 3, 5 и 6) можно выделив ее и щелкнув правой кнопкой мыши и выбрать пункт «Окно свойств». В выпадающем списке можно задать форму основной надписи используемой в рамке (верхняя строка на форме). Во второй строке на форме можно задать обозначение и наименование раздела, которое будет отображаться в форме. В данном случае используются названия и обозначения, применяемые в отрасли связи. Мастер «Спецификация» Этот мастер я задумал по причине того, что мне не нравилось как выглядит спецификация вставленная в Visio из Excel. Чтоб воспользоваться этим инструментом, нужно предварительно перенести свою спецификацию в Excel в файл спецификация.xls из архива. В этом файле подобраны высоты строк для отображения на первой странице 25 строк (случай на странице применяется Форма №3), на всех последующих страницах должно отражаться по 30 строк (случай на странице применяется Форма №6). Если вы применяете на странице Форму №5 ,то на странице должно умещаться 27 строк. Ширина столбцов также приближена к тому как это будет выглядеть на листе Visio. Есть два способа заполнения таблицы: 1. Предварительно скопировать нужное вам количество строк в Excel-евской таблице и вытащить мастер на лист. При этом появится вопрос окно диалога Paste special: из представленных вариантов необходимо выбрать Unformatted text или Unformatted Unicode text и нажать Ok. ┌───────────────────────────────────────────────────────────┐ │При копировании нужно ОБЯЗАТЕЛЬНО скопировать первый пустой столбец А │ └───────────────────────────────────────────────────────────┘ (см. образец файла в формате Excel в данном архиве) 2. Редактировать текст можно уже во вставленном мастере. Для этого можно нажать F2 или два раза щелкнуть мышью на таблице спецификации. Заполняем содержимое "ячейки" таблицы, для перехода к "ячейке" в следующем столбце нажать клавишу Tab на клавиатуре. Для перевода курсора для ввода текста на следующей строке нажимаем кнопку Enter. Для навигации по тексту пользуемся комбинацией Ctrl и стрелок навигации ←, ↑, ↓ и →. Ctrl+→ -Вправо Ctrl+← -Влево Ctrl+↑ -К началу предыдущей строки Ctrl+↓ -К началу следущей строки Табличные формы Общая информация К таким формам в данном комплекте относятся фигуры из наборов элементов ГОСТ Р 21.1101-2013 (формы 1, 2, 7 и 14), ГОСТ 21.501-2011 (формы 2-8) и ГОСТ Р 21.608-2014 (формы 1-3, 6). Все фигуры имеют специальное форматирование: с расстановкой табуляции текстовых полей (см. специальные символы над линейкой при редактировании блока текста) в соответсвующих столбцах таблицы; межстрочное расстояние 8 мм. Заполнение табличных форм производится по аналогии с заполнением фигуры Спецификация (см. описание выше). Аналогично возможна вставка скопированных значений из Excel, по аналогии с фигурой Спецификация из набора элементов ГОСТ Р 21.1101-2013. Для вызова контекстного меню Специальная вставка выделите табличную фигуру и щелкните правой кнопкой мыши. Выберите верхний пункт появившегося контекстного меню. Дополнительные возможности Некоторые их табличных фигур обладают особым поведением: 1. Позволяют изменять "высоту строки" табличной фигуры, потнув за контрол (желтую точку с правой стороны фигуры). Встречается в фигурах Форма 3 и Форма 6 (ГОСТ 21.501-2011. Приложение А) Для удобства подбора высоты кратной 8 мм, можно временно переключить размер сетки страницы на значение 8 мм. Для этого при выделенной табличной фигуре щелкните правой кнопкой мыши и в появившемся контекстном меню выберите пункт Установить сетку 8 мм. Переместив табличную фигуру так, чтобы нижняя часть шапки таблицы располагалась ровно на линии сетки. Полученную высоту строки можно увидеть задержав мышь над контролом на несколько секунд после перемещения. После установки желаемой "высоты строки", можно вернуть параметры сетки в исходное положение через команду Восстановить сетку в контекстном меню. 2. Изменять "ширину столбцов" таблицы, потнув за контрол (желтую точку в нижней части "шапки таблицы" фигуры). Встречается в фигурах Форма 3 и Форма 6 (ГОСТ 21.608-2014) Полученную ширину столбца можно увидеть задержав мышь над контролом на несколько секунд после перемещения.
