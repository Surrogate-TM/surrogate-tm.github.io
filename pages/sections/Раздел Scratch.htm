Scratch. Вспомогательный раздел

Рабочая область для ввода и проверки формул, на которые могут ссылаться другие ячейки.

Заметки

Этот раздел можно добавить в диалоговом окне Вставка раздела. Для этого щелкните окно таблицы свойств фигуры правой кнопкой мыши и выберите команду Вставить раздел.

Обычно вспомогательный раздел используется для изолирования сложных повторяющихся вычислений. Для вычислений с четко определенной целью проще и понятнее использовать ячейки раздела User-Defined Cells, поскольку им можно присвоить имена.

В ячейках вспомогательного раздела единицы используются двумя разными способами. В ячейках X и Y используются единицы документа; в ячейках A — D единицы не используются. (Ячейки X и Y на жаргоне программистов на C называются "заполненными", а ячейки с A по D — "пустыми".) Вспомогательные ячейки X и Y часто используются для производных координат от x и y, например ячейки PinX и PinY или ячейки X и Y в ячейке раздела Geometry. Во вспомогательных ячейках A — D отображаются единицы, определенные пользователем.

Также эти ячейки различаются способом хранения значений точки. В приложении Visio точка представляет собой единый пакет данных для координаты (x,y). При возврате значения точки формулой оно интерпретируется одним из трех способов в зависимости от того, какая формула введена в ячейке таблицы свойств фигуры. Ячейки, связанные с координатами x (например, ячейка PinX или ячейки в столбце X раздела Geometry), извлекают только координату x из значения точки. Ячейки, связанные с координатами y, извлекают только координату y из значения точки.
