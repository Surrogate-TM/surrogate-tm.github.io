Раздел Reviewer

Содержит сведения обо всех рецензентах документа.

Заметки

Раздел Reviewer содержит по одной строке для каждого рецензента, который добавил исправление в документ, даже если это исправление было впоследствии удалено.

Этот раздел находится в таблице свойств фигуры документа. Чтобы ее просмотреть, на вкладке Разработчик в группе Показать/скрыть выберите пункт Проводник по документам, затем щелкните документ правой кнопкой мыши и выберите команду Показать таблицу свойств фигуры.

Дополнительные сведения о примечаниях, добавляемых в документ как исправления, см. в разделе Annotation.
