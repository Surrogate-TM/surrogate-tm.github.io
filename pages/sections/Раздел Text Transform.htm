Раздел Text Transform

Здесь содержатся сведения о положении блока текста фигуры.

Заметки

Чтобы добавить этот раздел, откройте диалоговое окно Вставка раздела (щелкните правой кнопкой мыши область окна таблицы свойств фигуры и выберите команду Вставить раздел). Задание значений в разделе преобразования текста аналогично работе с блоком текста с помощью инструмента Блок текста (вкладка Главная).
