Раздел User-defined Cells

Этот раздел представляет собой рабочую область для ввода формул в пользовательские ячейки, на которые можно ссылаться в других ячейках и дополнительных компонентах.

Заметки

Как и вспомогательный раздел, раздел определенных пользователем ячеек можно использовать в собственных решениях различными способами. Преимущество этого раздела заключается в том, что ячейкам можно присваивать имена, а ячейки во вспомогательном разделе не могут быть именованными.

Если решение имеет четко определенное назначение, целесообразно использовать раздел пользовательских ячеек, поскольку этим ячейкам можно присваивать имена. Вспомогательный раздел обычно используется для повторяющихся сложных вычислений.

Чтобы добавить этот раздел, откройте диалоговое окно Вставка раздела (щелкните правой кнопкой мыши область окна таблицы свойств фигуры и выберите команду Вставить раздел).
