Раздел Layer Membership

Содержит одну строку с перечнем всех слоев, для которых назначена фигура.

Заметки

Назначение слоя отображается в качестве индекса к списку слоев страницы. Индекс слоя соответствует порядку слоев в диалоговом окне Свойства слоя (на вкладке Главная в группе Правка в меню Слои выберите команду Свойства слоя). Первое имя в диалоговом окне — слой 0, второе — слой 1 и т. д.

Если фигура назначена для нескольких слоев, индекс каждого слоя отображается в ячейке Layer Membership через точку с запятой.

Для ссылки на значение ячейки Layer Membership в формуле используйте имя LayerMember.
