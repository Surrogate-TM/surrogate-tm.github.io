Раздел Controls

Содержит строку и ячейки для каждого управляющего маркера, указанного для фигуры.

Заметки

Управляющие маркеры задаются в разделе Controls окна таблицы свойств фигуры. Если у фигуры нет этого раздела, его можно вставить в окне таблицы свойств фигуры. Для этого щелкните это окно правой кнопкой мыши, выберите команду Вставить раздел и установите флажок Элементы управления. Чтобы добавить управляющие маркеры в имеющийся раздел Controls, щелкните строку правой кнопкой мыши и в контекстном меню выберите команду Вставить строку.
