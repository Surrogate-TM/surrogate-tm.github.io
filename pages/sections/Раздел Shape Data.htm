Раздел Shape Data

Содержит по одной строке для каждого элемента данных фигуры и ячейки для связывания данных с фигурой.

Заметки

Данные фигуры можно задать в окне Данные фигуры или в ячейках раздела Shape Data.

  Примечание.

В предыдущих версиях приложения Visio этот раздел назывался Custom Properties.

         В окне таблицы свойств фигуры можно задать данные фигуры с помощью раздела Shape Data. Щелкните окно таблицы свойств фигуры правой кнопкой мыши, выберите команду Вставить раздел и установите флажок Данные фигуры. Можно добавить необходимое количество строк Prop.name, присвоить понятные имена строкам и задать значения ячеек.

         В окне документа можно задать данные фигуры, используя окно Данные фигуры. На вкладке Вид в группе Показать/скрыть выберите элемент Области задач, а затем — Данные фигуры. Чтобы открыть диалоговое окно Определение данных фигуры, щелкните окно Данные фигуры правой кнопкой мыши и выберите команду Определить данные фигуры.
