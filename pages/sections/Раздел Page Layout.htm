Раздел Page Layout

Содержит ячейки, которые отвечают за параметры страницы для фигур и соединительных линий, например за интервал между всеми фигурами или всеми соединительными линиями на странице и за стиль маршрута для всех соединительных линий на странице.

Заметки

Чтобы переопределить установленные по умолчанию параметры страницы для отдельной фигуры, используйте ячейки в разделе Shape Layout.

Значения для ячеек этого раздела можно задать в диалоговом окне Параметры страницы на вкладке Макеты и маршруты (на вкладке Разработка нажмите кнопку со стрелкой в группе Параметры страницы).
