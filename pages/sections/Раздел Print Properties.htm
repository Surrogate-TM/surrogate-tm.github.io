Раздел Print Properties

Содержит ячейки, которые отвечают за форматирование страницы документа при печати.

Заметки

Раздел Print Properties позволяет управлять печатью на уровне страницы. Настройки устанавливаются для каждой страницы и не зависят от настроек страницы документа. Например, страницу документа альбомной ориентации можно распечатать в книжной ориентации. Также можно печатать документы, используя комбинации ориентаций.

Настройки в разделе Print Properties соответствуют значениям, установленным в диалоговом окне Параметры печати на вкладке Параметры печати (на вкладке Разработка нажмите кнопку со стрелкой в группе Параметры печати). В разделе Page Properties можно просмотреть настройки, которые применяются к странице документа.
