Строка NURBSTo (раздел "Геометрия")

В этой строке содержатся координаты по осям X и Y, данные о положении предпоследнего узла, последней толщины, первого узла, первой толщины, а также формула для сплайна NURBS (nonuniform rational B-spline).

Ниже перечислены ячейки в строке NURBSTo.

Ячейка

Описание

X

Координата последней контрольной точки сплайна NURBS по оси X.

Y

Координата последней контрольной точки сплайна NURBS по оси Y.

A

Предпоследний узел сплайна NURBS.

B

Последняя толщина сплайна NURBS.

C

Первый узел сплайна NURBS.

D

Первая толщина сплайна NURBS.

E

Формула сплайна NURBS.

Заметки

Сплайн NURBS представляет собой основной способ математического представления кривой. Его можно создать с помощью инструмента Ломаная.
