Строка Ellipse (раздел "Геометрия")

В этой строке содержатся координаты центра эллипса по осям X и Y, а также координаты двух его точек.

Ниже перечислены ячейки в строке Ellipse.

Ячейка

Описание

X

Координата центра по оси X.

Y

Координата центра по оси Y.

A

Координата одной точки эллипса по оси X; в паре с координатой по оси Y, представленной в ячейке B.

B

Координата одной точки эллипса по оси Y; в паре с координатой по оси X, представленной в ячейке A.

C

Координата другой точки эллипса по оси X; в паре с координатой по оси Y, представленной в ячейке D.

D

Координата другой точки эллипса по оси Y; в паре с координатой по оси Y, представленной в ячейке C.

Заметки

В разделе "Геометрия" со строками Ellipse или InfiniteLine не должны содержаться другие строки.
