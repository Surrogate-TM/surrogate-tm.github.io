Строка MoveTo (раздел "Геометрия")

В этой строке содержатся координаты первой вершины фигуры по осям X и Y или координаты первой вершины после разрыва контура по осям X и Y.

Ниже перечислены ячейки в строке MoveTo.

Ячейка

Описание

X

Если строка MoveTo является первой в разделе, в этой ячейке содержится координата первой вершины фигуры по оси X.

Если строка MoveTo находится между двумя другими строками, в этой ячейке содержится координата первой вершины после разрыва контура по оси X.

Y

Если строка MoveTo является первой в разделе, в этой ячейке содержится координата первой вершины фигуры по оси Y.

Если строка MoveTo находится между двумя другими строками, в этой ячейке содержится координата первой вершины после разрыва контура по оси Y.

Заметки

Если строка MoveTo является первой в разделе, в ней содержатся координаты первой вершины фигуры по осям X и Y. Обычно это вершина, размещенная первой во время создания фигуры. Она не всегда соответствует начальной точке одномерной фигуры.

Строка MoveTo должна быть первой в разделе "Геометрия". Однако ее можно также использовать для обозначения разрыва в линии контура фигуры. Тем не менее, когда контур используется для определения границы области заливки, этот разрыв рассматривается как отрезок прямой линии. Чтобы вставить такой разрыв, вставьте строку между двух строк и измените ее тип на MoveTo. Если строка MoveTo находится между двух строк, в ней содержатся координаты первой вершины линии после разрыва контура фигуры по осям X и Y.
