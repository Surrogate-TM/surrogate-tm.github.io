Строка Hyperlink (раздел "Гиперссылки")

В этой строке содержатся данные об одной гиперссылке, связанной с фигурой. Для каждой гиперссылки в фигуре содержится одна строка Hyperlink.

Имена строк Hyperlink имеют вид Hyperlink.имя. Ниже перечислены ячейки, содержащиеся в этих строках. Дополнительные сведения см. в разделах, посвященных определенным ячейкам.

Ячейка

Описание

Description

Описательный текст для гиперссылки.

Address

URL-адрес, имя файла MS-DOS или UNC-путь для перехода.

SubAddress

Расположение в границах конечного документа, на который приводится ссылка.

ExtraInfo

Строка, содержащая данные для разрешения URL-адреса.

Frame

Имя конечной рамки при открытии приложения Microsoft Office Visio как активного документа в приложении-контейнере ActiveX. По умолчанию строка пуста.

SortKey

Определяет порядок отображения гиперссылок в контекстном меню.

NewWindow

Указывает, следует ли открывать гиперссылки в новом окне. Если значение равно TRUE, страница, документ или веб-сайт открывается в новом окне. По умолчанию используется значение FALSE.

Default

Определяет гиперссылку по умолчанию для фигуры или страницы.

Invisible

Указывает, следует ли отображать гиперссылку в контекстном меню.

Заметки

Можно добавлять любое количество строк Hyperlink.имя, присваивать им значимые имена, а также задавать значения для ячеек. Чтобы добавить гиперссылки в существующий раздел "Гиперссылки", щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Вставить строку.

Можно создавать ссылки на эти ячейки, используя имя строки, в которой они содержатся. Это имя отображается в окне ShapeSheet красным цветом. Чтобы присвоить строкам Hyperlink.имя значимые имена, щелкните строку и введите имя, например Маркетинг, чтобы создать строку Hyperlink.Маркетинг. Затем, используя имя Hyperlink.Маркетинг.Description, можно создать ссылку на ячейку Description.

Имя должно быть уникальным в пределах раздела.
