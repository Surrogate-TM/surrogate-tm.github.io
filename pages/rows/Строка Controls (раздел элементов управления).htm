Строка Controls (раздел "Элементов управления")

Содержит ячейки, определяющие координаты x и y и поведение каждого управляющего маркера, заданного для фигуры. Фигура содержит по одной строке элементов управления для каждого управляющего маркера.

Строки элементов управления именуются Controls.имя и содержат перечисленные ниже ячейки. Дополнительные сведение см. в разделах, описывающих конкретные ячейки.

Ячейка

Описание

x

Представляет координату x, соответствующую положению управляющего маркера фигуры в локальной системе координат.

y

Представляет координату y, соответствующую положению управляющего маркера фигуры в локальной системе координат.

X Dynamics

Представляет координату x точки привязки управляющего маркера в локальной системе координат.

Y Dynamics

Представляет координату y точки привязки управляющего маркера в локальной системе координат.

X Behavior

Управляет поведением координаты x управляющего маркера после его перемещения.

Y Behavior

Управляет поведением координаты y управляющего маркера после его перемещения.

Can Glue

Определяет, можно ли приклеить управляющий маркер к другим фигурам.

Tip

Представляет строку с описательным текстом, которая появляется в виде всплывающей подсказки при наведении курсора на управляющий маркер фигуры.

Примечания

Можно добавлять неограниченное количество строк Controls.имя, присваивать им осмысленные имена и задавать значения ячеек. Чтобы добавить маркер управления в существующий раздел элементов управления, щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Вставить строку.

На ячейки можно ссылаться по имени строки, которое отображается красным цветом в окне таблицы свойств фигуры. Чтобы назначить строке Controls.имя осмысленное имя, щелкните ее и введите, например, имя Custom, чтобы создать имя строки Controls.Custom. После этого можно ссылаться на ячейку X как на Controls.Custom.X.

Вводимое имя строки должно быть уникальным в пределах раздела.
