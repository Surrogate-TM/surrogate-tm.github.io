Строка Connection Points (раздел "Точек соединения")

Содержит координаты x и y, направление по вертикали и горизонтали и тип для одной точки соединения фигуры. Координаты точек соединения отсчитываются от начала координат фигуры. Фигура содержит одну строку точек соединения для каждой такой точки.

Если строки точек соединения именованные, их имена отображаются в виде Connections.имя в окне таблицы свойств фигуры. Строки точек соединения содержат перечисленные ниже ячейки. Дополнительные сведения см. в разделах, описывающих конкретные ячейки.

Ячейка

Описание

X

Координата x точки соединения в локальной системе координат.

Y

Координата y точки соединения в локальной системе координат.

DirX/A

Компонент x необходимого вектора выравнивания для соответствующей точки соединения. Ячейка DirX/A также используется для того, чтобы задать направление прикрепленной перекладины динамической соединительной линии. Эта ячейка содержит значение с плавающей запятой.

DirY/B

Компонент y необходимого вектора выравнивания для соответствующей точки соединения. Ячейка DirY/B также используется для того, чтобы задать направление прикрепленной перекладины динамической соединительной линии. Эта ячейка содержит значение с плавающей запятой.

Type/C

Тип точки соединения (0 — внутрь, 1 — наружу, 2 — внутрь и наружу).

D

Вспомогательная ячейка, которую можно использовать для ввода и проверки формул. Чтобы обратиться к этой ячейке, щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Изменить тип строки.

Примечания

Ячейки в строке Connections.имя называются DirX/A, DirxY/B и Type/C, поскольку соответствующие строки могут быть развернутыми или неразвернутыми.

Большинство точек соединения (все точки, созданные с помощью пользовательского интерфейса) являются неразвернутыми и содержат ячейки DirX, DirY и Type. Эти строки имеют тип visTagCnnctPt или visTagCnnctNamed.

В неразвернутых строках ячейки DirX и DirY определяют вектор направления, который влияет на вращение фигур, связанных с помощью точки соединения. Если обе координаты равны нулю, точка не имеет направления. Точки соединения могут иметь перечисленные ниже типы.

         Внутрь (0). К таким точкам приклеиваются фигуры. Это значение по умолчанию.

         Наружу (1). Такие точки приклеиваются к точкам соединения типа "внутрь".

         Внутрь и наружу (2). В этом случае для точки задано направление "внутрь", но при ее использовании в соединении, направленном наружу, направление изменяется на противоположное.

Развернутые строки содержат ячейки A, B, C и D и ведут себя как ненаправленные неразвернутые строки типа "внутрь". Развернутые строки используются редко, однако с их помощью можно связать данные с точкой соединения в ячейках A, B, C и D. Такие строки имеют тип visTagCnnctPtABCD или visTagCnnctNamedABCD. Развернутые строки можно идентифицировать по наличию формулы в ячейке D.

Можно добавлять неограниченное количество строк Connections.имя, присваивать им осмысленные имена и задавать значения ячеек. Чтобы добавить точку соединения в существующий раздел точек соединения, щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Вставить строку.

На ячейки строк точек соединения можно ссылаться по имени строки, которое отображается красным цветом в окне таблицы свойств фигуры. Чтобы изменить имя строки, щелкните ее и введите, например, имя Custom, чтобы создать имя строки Connections.Custom. После этого можно ссылаться на ячейку X по имени Connections.Custom.X или Connections.X1 (в последнем случае используется номер строки).

Введенное имя строки должно быть уникальным в пределах раздела. При создании имени для одной из строк в разделе точек соединения Microsoft Office Visio назначает всем строкам раздела имена по умолчанию: Connections.Row_n.

Именованные строки точек соединения несовместимы с более ранними версиями Visio, чем 5.0. При сохранении файла документа Visio с именованными строками точек соединения в формате более ранней версии ссылки на именованные строки точек соединения преобразуются в индексированные ссылки, а имена строк теряются.
