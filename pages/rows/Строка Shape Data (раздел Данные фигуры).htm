Строка Shape Data (раздел "Данные фигуры")

В этой строке содержатся данные об одном элементе данных фигуры, связанном с фигурой. Фигура содержит одну строку Shape Data для каждого элемента данных.

Имена строк Shape Data имеют вид Prop.имя. Ниже перечислены ячейки, содержащиеся в этих строках. Дополнительные сведения см. в разделах, посвященных определенным ячейкам.

Ячейка

Описание

Label

Задает подпись, которая отображается в диалоговом окне Данные фигуры.

Prompt

Задает описательный или пояснительный текст, который отображается при выборе элемента в диалоговом окне Данные фигуры.

Type

Задает тип данных для значения элемента данных фигуры: строка (0), постоянный список (1), число (2), логическое значение (3), переменный список (4), дата или время (5), длительность (6) или денежная единица (7).

Format

Задает формат элемента данных фигуры.

Value

Содержит значение элемента, указанное в диалоговом окне Данные фигуры.

SortKey

Задает ключ, согласно которому упорядочиваются элементы в диалоговом окне Данные фигуры.

Invisible

Указывает, отображается ли элемент данных фигуры в диалоговом окне Данные фигуры. TRUE — не отображается, FALSE — отображается.

Ask

Определяет, требуется ли пользователю указывать данные фигуры при создании экземпляра или дублировании и копировании фигуры.

LangID

Определяет язык, на котором отображается значение элемента данных фигуры.

Calendar

Определяет тип используемого календаря, если в качестве типа элемента данных фигуры задана дата.

Заметки

Можно добавлять любое количество строк Prop.имя, присваивать им значимые имена, а также задавать значения для ячеек. Чтобы добавить элемент данных фигуры в существующий раздел "Данные фигуры", щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Вставить строку.

Можно создавать ссылки на эти ячейки, используя имя строки, в которой они содержатся. Это имя отображается в окне ShapeSheet красным цветом. Чтобы присвоить строкам Prop.имя значимые имена, щелкните строку и введите имя, например Цена, чтобы создать строку Prop.Цена. Затем, используя имя Prop.Цена.Label, можно создать ссылку на ячейку Label.

Имя должно быть уникальным в пределах раздела.
