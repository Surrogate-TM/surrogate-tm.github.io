Строка User-defined Cells (раздел "Пользовательские ячейки")

В этой строке содержатся значение и описательная подсказка для пользовательских ячеек. Фигура содержит одну строку User-defined Cells для каждой пары пользовательских ячеек Value/Prompt.

Имена строк User-defined Cells имеют вид User.имя. Ниже перечислены ячейки, содержащиеся в этих строках. Дополнительные сведения см. в разделах, посвященных определенным ячейкам.

Ячейка

Описание

Value

Задает значение для соответствующей пользовательской ячейки.

Prompt

Задает описательную подсказку или комментарий для пользовательской ячейки.

Заметки

В пользовательские ячейки можно вводить формулы или константы, на которые могут ссылаться другие ячейки и надстройки. Значения в пользовательских ячейках можно переносить. Например, при копировании фигуры, ссылающейся на пользовательскую ячейку одной фигуры, в другую фигуру, в которой нет такой же пользовательской ячейки, такая ячейка добавляется в фигуру.

Можно добавлять любое количество строк User.имя, присваивать им значимые имена, а также задавать значения для ячеек. Чтобы добавить строку в существующий раздел "Пользовательские ячейки", щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Вставить строку.

Можно создавать ссылки на эти ячейки, используя имя строки, в которой они содержатся. Это имя отображается в окне ShapeSheet красным цветом. Чтобы присвоить строкам User.имя значимые имена, щелкните строку и введите имя, например Смещение, чтобы создать строку User.Смещение. Затем, используя имя User.Смещение.Prompt, можно создать ссылку на ячейку Prompt.

Имя должно быть уникальным в пределах раздела.
