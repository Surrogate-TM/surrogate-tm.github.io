Строка Actions (раздел "Действий")

Содержит ячейки, которые определяют действия, связанные с настраиваемой командой в контекстном меню или меню тегов действий. Раздел действий содержит по одной строке действий для каждого действия.

  Примечание.

В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами.

Строки действий именуются Actions.имя и содержат перечисленные ниже ячейки. Дополнительные сведения см. в разделах, описывающих конкретные ячейки.

Ячейка

Описание

Action

Содержит формулу, которая выполняется, когда пользователь выбирает элемент в контекстном меню или меню тегов действий.

Menu

Определяет имя элемента меню тегов действий или контекстного меню.

TagName

Логическое имя тега действия, в котором должно отображаться указанное действие.

ButtonFace

Задает значок, отображаемый рядом с элементом в контекстном меню или в меню тегов действий.

SortKey

Число, определяющее порядок элементов в меню тегов действий или контекстном меню.

Checked

Указывает, должен ли пункт меню тегов действий или контекстного меню быть отмечен галочкой.

Disabled

Указывает, отключен ли элемент меню тегов действий или контекстного меню.

ReadOnly

Указывает, является ли элемент меню элементом только для чтения (элементом, который нельзя выбрать).

Invisible

Указывает, является ли элемент меню тегов действий или контекстного меню видимым.

BeginGroup

Указывает, нужно ли вставлять в меню над этим пунктом разделитель.

Примечания

Можно добавлять неограниченное количество строк Actions.имя, присваивать им осмысленные имена и задавать значения ячеек. Чтобы добавить настраиваемую команду в существующий раздел действий, щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Вставить строку.

На ячейки можно ссылаться по имени строки, которое отображается красным цветом в окне таблицы свойств фигуры. Чтобы назначить строке Actions.имя осмысленное имя, щелкните ее и введите, например, имя Custom, чтобы создать имя строки Actions.Custom. После этого можно ссылаться на ячейку Menu как на Actions.Custom.Menu.

Вводимое имя строки должно быть уникальным в пределах раздела.
