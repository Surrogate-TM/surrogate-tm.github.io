Строка Smart Tags (раздел "Теги действий")

В этой строке содержатся данные об одном теге действия, связанном с фигурой или страницей. Фигура или страница содержит одну строку Smart Tags для каждого тега действия.

  Примечание.

В предыдущих версиях приложения Microsoft Visio теги действий называются смарт-тегами.

Имена строк Smart Tags имеют вид SmartTags.имя. Ниже перечислены ячейки, содержащиеся в этих строках. Дополнительные сведения см. в разделах, посвященных определенным ячейкам.

Ячейка

Описание

X

Координата точки по оси X в локальных координатах фигуры, вокруг которой размещается кнопка тега действия.

Y

Координата точки по оси Y в локальных координатах фигуры, вокруг которой размещается кнопка тега действия.

TagName

Логическое имя тега действия.

X Justify

Смещение кнопки тега действия по оси X относительно точки, заданной ячейками X и Y.

Y Justify

Смещение кнопки тега действия по оси Y относительно точки, заданной ячейками X и Y.

DisplayMode

Определяет время появления тега действия.

ButtonFace

Идентификатор изображения, которое будет использоваться для кнопки тега действия.

Description

Строка описания тега действия.

Disabled

Указывает, отключен ли тег действия.

Заметки

Можно добавлять любое количество строк SmartTags.имя, присваивать им значимые имена, а также задавать значения для ячеек. Чтобы добавить тег действия в существующий раздел "Смарт-теги", щелкните строку правой кнопкой мыши и выберите в контекстном меню команду Вставить строку.

Можно создавать ссылки на эти ячейки, используя имя строки, в которой они содержатся. Это имя отображается в окне ShapeSheet красным цветом. Чтобы присвоить строкам Smart Tags.имя значимые имена, щелкните строку и введите имя, например Размер, чтобы создать строку Smart Tags.Размер. Затем, используя имя SmartTags.Размер.Description, можно создать ссылку на ячейку Description.

Имя должно быть уникальным в пределах раздела.
