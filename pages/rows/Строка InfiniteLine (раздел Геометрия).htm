Строка InfiniteLine (раздел "Геометрия")

В этой строке содержатся координаты двух точек на бесконечной линии по осям X и Y.

Ниже перечислены ячейки в строке InfiniteLine.

Ячейка

Описание

X

Координата точки на бесконечной линии по оси X; в паре с координатой по оси Y, представленной в ячейке Y.

Y

Координата точки на бесконечной линии по оси Y; в паре с координатой по оси X, представленной в ячейке X.

A

Координата точки на бесконечной линии по оси X; в паре с координатой по оси Y, представленной в ячейке B.

B

Координата точки на бесконечной линии по оси Y; в паре с координатой по оси X, представленной в ячейке A.

Заметки

В разделе "Геометрия" со строками Ellipse или InfiniteLine не должны содержаться другие строки.
