Добро пожаловать в справочник по таблице свойств фигуры в Microsoft Visio 2010

В справочнике содержатся сведения об основных понятиях, описания задач программирования, примеры и ссылки, упрощающие разработку решений на основе Microsoft Visio 2010.

Мы ждем ваших отзывов и вопросов по этому материалу. Отзыв можно отправить по адресу docthis@microsoft.com.

Дата публикации справочника: март 2010 г.

Различные версии данной документации можно получить из перечисленных ниже источников.

    Из программного продукта (новейшая версия).  Если компьютер подключен к Интернету, новейшую версию данного справочника можно открыть из приложения Visio. Нажмите в приложении Visio кнопку Справка и щелкните стрелку раскрывающегося списка Поиск. В разделе Контент с сайта Office.com выберите пункт Справочник разработчика.

    Из программного продукта (установленная версия).  Если подключение к Интернету отсутствует, можно воспользоваться версией справочника, поставляемой вместе с продуктом. Нажмите в приложении Visio кнопку Справка и щелкните стрелку раскрывающегося списка Поиск. В разделе Контент с данного компьютера выберите пункт Справочник разработчика.

    В Центре загрузки Майкрософт.  Чтобы загрузить новейшую версию справочника по таблице свойств фигуры для автономного просмотра, перейдите на страницу пакета SDK для Visio 2010 Центра загрузки Майкрософт (на английском языке).

    В библиотеке MSDN.  Чтобы просмотреть новейшую версию справочника по таблице свойств фигуры в Visio 2010 с помощью браузера, откройте справочник на сайте библиотеки MSDN (на английском языке).

Данная документация содержит указанные ниже разделы.

    Новые возможности: список новых членов по объектам и в алфавитном порядке.

    Основные понятия: описание основных понятий, необходимых для разработки пользовательских решений на основе Visio.

    Справочник: по таблице свойств фигуры в Visio.
