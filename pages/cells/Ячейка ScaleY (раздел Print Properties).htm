Ячейка ScaleY (раздел "Print Properties")

Задает процент увеличения страницы документа на печатной странице.

Заметки

Это значение используется только в том случае, если значение ячейки OnPage равно FALSE. Ячейки ScaleX и ScaleY всегда имеют одно и то же значение, которое соответствует значению параметра Установить на вкладке Настройка печати диалогового окна Параметры страницы (на вкладке Конструктор щелкните стрелку Параметры страницы).

Чтобы создать ссылку на ячейку ScaleY по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ScaleY

Чтобы создать ссылку на ячейку ScaleY из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowPrintProperties

Индекс ячейки:

visPrintPropertiesScaleY
