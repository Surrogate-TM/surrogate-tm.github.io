Ячейка DirY / B (раздел Connection Points)

Определяет компонент Y необходимого вектора выравнивания для соответствующей точки соединения. Ячейка DirY / B также используется для указания направления прикрепленной перекладины динамической соединительной линии. Эта ячейка содержит значение с плавающей запятой.

Примечания

Чтобы получить ссылку на ячейку DirY / B по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

DirY[i],
где i = <1>, 2, 3...

Чтобы получить ссылку на ячейку DirY / B по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionConnectionPts

Индекс строки:

visRowConnectionPts+ i,
где i = 0, 1, 2...

Индекс ячейки:

visCnnctDirY (для неразвернутых строк)
visCnnctB (для развернутых строк)

Сведения о развернутых и неразвернутых строках см. в разделе "Строка точек соединения".
