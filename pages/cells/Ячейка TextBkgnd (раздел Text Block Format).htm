Ячейка TextBkgnd (раздел "Text Block Format")

Определяет цвет фона текста для фигуры.

Заметки

Ячейка TextBkgnd может содержать любое значение от 0 до 24 или значение 255. Значения 0 и 255 (visTxtBlklOpaque) обозначают прозрачный фон текста.

Чтобы ввести пользовательский цвет, используйте функцию RGB или HSL плюс 1, например RGB(255,127,255)+1. Значением пользовательского цвета является цвет RGB, и в окне таблицы свойств фигуры будет отображаться RGB(r, g, b)+1, а не число. При использовании в числовых операциях пользовательским цветам назначаются значения 25 и выше.

Можно задать прозрачность цвета фона текста с помощью ячейки TextBkgndTrans.

Чтобы создать ссылку на ячейку TextBkgnd по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

TextBkgnd

Чтобы создать ссылку на ячейку TextBkgnd из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowText

Индекс ячейки:

visTxtBlkBkgnd
