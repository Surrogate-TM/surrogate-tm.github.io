Ячейка ShdwType (раздел "Page Properties")

Указывает тип тени по умолчанию для страницы.

Значение

Описание

Константа автоматизации

1

Простая

visFSTSimple

2

Наклонная

visFSTOblique

Заметки

Тип тени, указанный в этой ячейке, используется, когда для ячейки ShapeShdwType (тип тени для отдельной фигуры на странице) задано значение "Использовать параметры страницы, заданные по умолчанию" (visFSTPageDefault).

Типы простых теней называются в интерфейсе тенями со смещением. Простая тень создает впечатление тени фигуры, спроецированной на параллельную плоскость, расположенную на определенном расстоянии за ней. Наклонные тени имеют в интерфейсе такое же название и создают впечатление тени, спроецированной на плоскость, перпендикулярную фигуре.

Список готовых типов простых и наклонных теней см. в поле Стиль на вкладке Тени диалогового окна Параметры страницы (на вкладке Конструктор щелкните стрелку Параметры страницы).

Чтобы создать ссылку на ячейку ShdwType по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShdwType

Чтобы создать ссылку на ячейку ShdwType из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowPage

Индекс ячейки:

visPageShdwType
