Ячейка ShdwScaleFactor (раздел "Page Properties")

Определяет процентное соотношение для увеличения или уменьшения тени фигуры.

Заметки

У каждой тени есть затененный центр вращения, то есть точка, соответствующая центру вращения фигуры. Например, если центр вращения фигуры находится в центре фигуры, затененный центр вращения будет точкой в центре тени. При применении масштаба к простым теням увеличение выполняется относительно затененного центра вращения. В случае с тенью, расположенной с наклоном, увеличение применяется к наклону.

Этот процент используется, когда для типа тени фигуры задано значение "Использовать параметры страницы, заданные по умолчанию" (ячейка ShapeShdwType равна visFSTPageDefault).

Чтобы настроить это поведение для отдельной фигуры, используйте ячейку ShapeShdwScaleFactor в разделе "Fill Format".

Это значение соответствует значению параметра Увеличение на вкладке Тени диалогового окна Параметры страницы (на вкладке Конструктор щелкните стрелку Параметры страницы).

Чтобы создать ссылку на ячейку ShdwScaleFactor по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShdwScaleFactor

Чтобы создать ссылку на ячейку ShdwScaleFactor из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowPage

Индекс ячейки:

visPageShdwScaleFactor
