Ячейка Ask (раздел Shape Data)

Определяет, предлагается ли пользователю ввести данные фигуры при создании ее экземпляра, дублировании или копировании.

Значение

Описание

TRUE

Пользователю предлагается ввести данные фигуры в диалоговом окне Определеные данных фигуры.

FALSE

Пользователю не предлагается ввести данные.

Заметки

Значение в этой ячейке соответствует значению флажка Спрашивать при размещении в диалоговом окне Определение данных фигуры (щелкните фигуру правой кнопкой мыши, наведите указатель мыши на пункт Данные, а затем щелкните элемент Определить данные фигуры).

Чтобы получить ссылку на ячейку Ask по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Prop.имя.Ask,
где Prop.имя — имя строки пользовательского свойства

Чтобы получить ссылку на ячейку Ask по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionProp

Индекс строки:

visRowProp + i,
где i = 0, 1, 2,...

Индекс ячейки:

visCustPropsAsk
