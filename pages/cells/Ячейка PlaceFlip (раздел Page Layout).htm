Ячейка PlaceFlip (раздел Page Layout)

Определяет порядок отражения и поворота размещаемых фигур на странице при использовании диалогового окна Настройка макета (чтобы вызвать его, на вкладке Конструктор в группе Макет выберите команду Изменить макет страницы, а затем — Дополнительные параметры макета).

Значение

Описание

Вариант по умолчанию. Отражение не выполняется.

visLOFlipDefault

Заметки

Значение ячейки PlaceFlip помогает расположить фигуру нужной стороной относительно связанной фигуры, размещенной рядом. Обычно эта возможность используется при составлении документов со статическим склеиванием.

Чтобы настроить это поведение для конкретной фигуры, используйте ячейку ShapePlaceFlip в разделе Shape Layout.

Чтобы получить ссылку на ячейку PlaceFlip по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

PlaceFlip

Чтобы получить ссылку на ячейку PlaceFlip по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowPageLayout

Индекс ячейки:

visPLOPlaceFlip
