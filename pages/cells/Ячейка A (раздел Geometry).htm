Ячейка A (раздел Geometry)

Представляет разные сведения в разных строках. В приведенной ниже таблице описаны значения ячейки A в зависимости от того, в какой строке она расположена.

Строка

Описание

ArcTo

Расстояние от середины дуги до середины ее хорды.

EllipticalArcTo

Координата X контрольной точки дуги (точки на дуге). Оптимальное расположение контрольной точки находится в районе середины пути между начальной и конечной точками. В противном случае дуга может принять слишком большой размер, чтобы пройти через контрольную точку, что может привести к непредсказуемым результатам.

PolylineTo

Формула ломаной линии.

NURBSTo

Предпоследний узел неравномерного рационального B-сплайна (NURBS-сплайна).

SplineStart

Второй узел сплайна.

SplineKnot

Один из узлов сплайна (не последний и не первые два).

InfiniteLine

Координата X точки на бесконечной линии; используется в паре с координатой Y, которая указана в ячейке B.

Ellipse

Координата X точки на эллипсе; используется в паре с координатой Y, которая указана в ячейке B.

Примечания

Чтобы получить ссылку на ячейку A по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Geometryi.Aj,
где i и j = <1>, 2, 3...

Geometryi.A1 (строки InfiniteLine и Ellipse),
где i = <1>, 2, 3...

Чтобы получить ссылку на ячейку A по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionFirstComponent + i,
где i = 0, 1, 2...

Индекс строки:

visRowVertex + j,
где j = 0, 1, 2...

visRowVertex (строки InfiniteLine и Ellipse)

Индекс ячейки:

visBow (строка ArcTo)

visControlX (строка EllipticalArcTo)

visControlY (строка EllipticalArcTo)

visPolylineData (строка Polyline)

visNURBSKnot (строка NURBSTo)

visSplineKnot (строки SplineStart и SplineKnot)

visInfiniteLineX2 (строка InfiniteLine)

visEllipseMajorX (строка Ellipse)
