Ячейка FlipX (раздел Shape Transform)

Указывает, была ли фигура отражена по горизонтали.

Значение

Описание

TRUE

Фигура была отражена по горизонтали.

FALSE

Фигура не была отражена по горизонтали.

Примечания

Чтобы получить ссылку на ячейку FlipX по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

FlipX

Чтобы получить ссылку на ячейку FlipX по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowXFormOut

Индекс ячейки:

visXFormFlipX
