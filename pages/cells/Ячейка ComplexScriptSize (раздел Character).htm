Ячейка ComplexScriptSize (раздел Character)

Размер шрифта, используемого для форматирования текста с символами из набором сложных знаков.

Заметки

Размеры шрифтов для наборов сложных знаков указаны на вкладке Шрифт в диалоговом окне Текст (щелкните стрелку в группе Шрифт на вкладке Главная). Этот список отображается только в том случае, если с помощью диалогового окна Языковые параметры Microsoft Office был добавлен язык, содержащий символы азиатских языков или символы из наборов сложных знаков (чтобы его вызвать, нажмите кнопку Пуск и выберите по очереди пункты Все программы, Microsoft Office, Средства Microsoft Office и Языковые параметры Microsoft Office.

Это значение можно задать явным образом как размер в точках либо как процентную долю. Процентная доля рассчитывается относительно значения в ячейке Size. Значение по умолчанию 0 (нуль) соответствует 100%.

Чтобы получить ссылку на ячейку ComplexScriptSize по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже аргументы.

Имя ячейки:

Char.ComplexScriptSize[i],
где i = <1>, 2, 3...

Чтобы получить ссылку на ячейку ComplexScriptSize по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionCharacter

Индекс строки:

visRowCharacter + i,
где i = 0, 1, 2...

Индекс ячейки:

visCharacterComplexScriptSize
