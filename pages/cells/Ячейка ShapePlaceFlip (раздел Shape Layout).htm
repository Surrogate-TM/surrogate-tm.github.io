Ячейка ShapePlaceFlip (раздел "Shape Layout")

Определяет способ отражения и поворота размещаемой фигуры на странице при размещении фигур с помощью диалогового окна Настройка макета (на вкладке Конструктор в группе Макет щелкните Изменить расположение фигур, а затем выберите пункт Дополнительные параметры разметки).

Значение

Описание

Константа автоматизации

0

Использовать параметры страницы, заданные по умолчанию.

visLOFlipDefault

1

Отразить слева направо.

visLOFlipX

2

Отразить сверху вниз.

visLOFlipY

4

Отразить на угол от 0 до 270 градусов с шагом в 90 градусов.

visLOFlipRotate

8

Не отражать.

visLOFlipNone

Заметки

Это значение ячейки ShapePlaceFlip позволяет сориентировать размещаемую фигуру относительно следующей размещаемой фигуры, с которой она соединена.

Чтобы задать это поведение для всех фигур на странице документа, используйте ячейку PlaceFlip в разделе "Page Layout".

Чтобы создать ссылку на ячейку ShapePlaceFlip по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapePlaceFlip

Чтобы создать ссылку на ячейку ShapePlaceFlip из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:

visSLOPlaceFlip
