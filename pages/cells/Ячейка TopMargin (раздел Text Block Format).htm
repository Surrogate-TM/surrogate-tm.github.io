Ячейка TopMargin (раздел "Text Block Format")

Определяет расстояние между верхней границей блока текста и первой строкой содержащегося в нем текста. По умолчанию установлено значение 4,0000 пт. Это значение не зависит от масштаба документа. Если масштаб документа изменяется, верхнее поле остается прежним.

Примечания

Чтобы создать ссылку на ячейку TopMargin по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

TopMargin

Чтобы создать ссылку на ячейку TopMargin из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowText

Индекс ячейки:

visTxtBlkTopMargin
