Ячейка LineJumpFactorY (раздел Page Layout)

Определяет размер значков пересечения на вертикальных динамических соединительных линиях страницы относительно значения в ячейке LineToLineY. Значение в этой ячейке может находиться в диапазоне от 0 до 10, однако рекомендуется использовать дробные значения от 0 до 1.

Заметки

Значение этой ячейки также можно задать на вкладке Макеты и маршруты диалогового окна Параметры страницы (на вкладке Конструктор щелкните стрелку Параметры страницы, а затем — элемент Макеты и маршруты).

Чтобы получить ссылку на ячейку LineJumpFactorY по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

LineJumpFactorY

Чтобы получить ссылку на ячейку LineJumpFactorY по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowPageLayout

Индекс ячейки:

visPLOJumpFactorY
