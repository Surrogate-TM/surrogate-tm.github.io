Ячейка ObjType (раздел Miscellaneous)

Определяет, являются ли объекты на схемах размещаемыми или направляемыми при создании макетов схем с помощью диалогового окна Настройка макета.

Значение

Описание

Константа автоматизации

&H0

Вариант по умолчанию. Приложение само принимает решение на основе контекста документа.

visLOFlagsVisDecides

&H1

Фигура является размещаемой.

visLOFlagsPlacable

&H2

Фигура является направляемой. Она должна быть одномерной.

visLOFlagsRoutable

&H4

Фигура не является ни размещаемой, ни направляемой.

visLOFlagsDont

&H8

Группа содержит размещаемые и направляемые фигуры.

visLOFlagsPNRGroup

Заметки

По умолчанию в ячейке ObjType для фигуры задается значение No Formula, возвращающее 0. Это означает, что приложение само определяет, является ли фигура размещаемой, исходя из контекста. Например, для простого прямоугольника значением ячейки ObjType будет 0. Если после этого соединить прямоугольник с другой фигурой с помощью элемента Соединительная линия, Visio задаст для ячейки ObjType данного прямоугольника значение 1 (размещаемая фигура).

Значением ячеки ObjType может быть сочетание значений. Если установлен бит неразмещаемой фигуры (&H4), он имеет приоритет над другими значениями, кроме значения группы (&H8).

Чтобы получить ссылку на ячейку ObjType по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

ObjType

Чтобы получить ссылку на ячейку ObjType по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowMisc

Индекс ячейки:
