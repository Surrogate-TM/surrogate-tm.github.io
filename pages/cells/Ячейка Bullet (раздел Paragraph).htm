Заметки

Чтобы задать значение этой ячейки, можно также щелкнуть фигуру правой кнопкой мыши, навести указатель на пункт Формат, выбрать пункт Текст и открыть вкладку Маркеры.

Чтобы получить ссылку на ячейку Bullet по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Para.Bullet[i],
где i = <1>, 2, 3, ...

Чтобы получить ссылку на ячейку Bullet по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionParagraph

Индекс строки:

visRowParagraph + i,
где i = 0, 1, 2, ...

Индекс ячейки:

visBulletIndex
