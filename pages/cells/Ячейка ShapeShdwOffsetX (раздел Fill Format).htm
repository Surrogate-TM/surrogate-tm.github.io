Ячейка ShapeShdwOffsetX (раздел "Fill Format")

Определяет расстояние смещения тени фигуры по горизонтали относительно фигуры в единицах страницы.

Заметки

Это значение соответствует значению параметра Смещение по X диалогового окна Тень (на вкладке Главная в группе Фигура выберите пункт Тень и командуПараметры тени).

Чтобы создать ссылку на ячейку ShapeShdwOffsetX по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapeShdwOffsetX

Чтобы создать ссылку на ячейку ShapeShdwOffsetX из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowFill

Индекс ячейки:

visFillShdwOffsetX
