Ячейка ShapeSplittable (раздел "Shape Layout")

Указывает, является ли одномерная фигура разделяемой.

Значение

Описание

Константа автоматизации

0

Фигура не может разделяться.

visSLOSplittableNone

1

Фигура может разделяться.

visSLOSplittableAllow

Заметки

Поведение, заданное по умолчанию для соединителей и других одномерных фигур, зависит от типа документа.

Автоматическое разделение фигур включается или отключается на трех различных уровнях: на уровне приложения, страницы и фигуры. По умолчанию разделение включено на уровнях приложения и страницы.

Чтобы включить или отключить разделение на уровне приложения, используйте параметр Разрешить разделение соединительной линии на вкладке Дополнительно диалогового окна Параметры Visio (откройте вкладку Файл, выберите пункт Параметры и вкладку Дополнительно).

Чтобы включить или отключить разделение на странице, используйте ячейку PageShapeSplit.

Чтобы включить разделение одномерной фигуры, допускающей разделение, используйте ячейку ShapeSplit.

Чтобы создать ссылку на ячейку ShapeSplittable по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapeSplittable

Чтобы создать ссылку на ячейку ShapeSplittable из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:

visSLOSplittable
