Ячейка IsTextEditTarget (раздел Group Properties)

Определяет порядок назначения текста для группы.

Значение

Описание

TRUE

Текст добавляется в фигуру группы.

FALSE

Текст добавляется в фигуру, расположенную в группе поверх остальных.

Заметки

Чтобы задать это значение, можно также выбрать группу, открыть вкладку Разработчик, щелкнуть элемент Поведение и установить флажок Изменить текст группы.

Для групп, созданных в версиях Visio до 2000, по умолчанию задается значение FALSE. Начиная с Visio 2000, значение по умолчанию — TRUE.

Чтобы получить ссылку на ячейку IsTextEditTarget по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

IsTextEditTarget

Чтобы получить ссылку на ячейку IsTextEditTarget по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowGroup

Индекс ячейки:

visGroupIsTextEditTarget
