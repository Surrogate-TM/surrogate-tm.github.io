Ячейка TheText (раздел "Events")

Ячейка события, рассчитываемая при изменении текста фигуры или его композиции.

Примечания

Ячейки событий вычисляются только при возникновении события, а не при вводе формулы. С помощью ячейки TheText можно вызывать повторные вычисления, например повторное вычисление ширины и высоты текста с помощью функций TEXTWIDTH( ) и TEXTHEIGHT( ).

Чтобы создать ссылку на ячейку TheText по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

TheText

Чтобы создать ссылку на ячейку TheText из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowEvent

Индекс ячейки:

visEvtCellTheText
