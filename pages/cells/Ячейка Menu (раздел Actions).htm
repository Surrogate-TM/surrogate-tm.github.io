Ячейка Menu (раздел Actions)

Определяет имя пункта в контекстном меню или меню тегов действий фигуры или страницы.

  Примечание.

В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами.

Заметки

Чтобы вставить в меню разделитель над данным пунктом, используйте ячейку BeginGroup. Чтобы отобразить команду внизу меню, укажите перед ее именем символ процента (%) .

Чтобы получить ссылку на ячейку Menu по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Actions.имя.Menu,

где Actions.имя — имя строки действий

Чтобы получить ссылку на ячейку Menu по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionAction

Индекс строки:

visRowAction + i,

где i = 0, 1, 2, ...

Индекс ячейки:

visActionMenu
