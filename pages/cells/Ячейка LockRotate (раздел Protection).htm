Ячейка LockRotate (раздел Protection)

Запрещает поворот двумерной фигуры с помощью маркера поворота или команд Повернуть влево на 90° и Повернуть вправо на 90°.

Значение

Описание

TRUE

Фигура не может быть повернута.

FALSE

Фигура может быть повернута (вариант по умолчанию).

Примечания

Ячейка LockRotate не запрещает поворот одномерной фигуры при перетаскивании ее за конечную точку. Чтобы запретить поворот одномерной фигуры, задайте для ячейки LockWidth ненулевое значение (TRUE).

Чтобы получить ссылку на ячейку LockRotate по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

LockRotate

Чтобы получить ссылку на ячейку LockRotate по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowLock

Индекс ячейки:

visLockRotate
