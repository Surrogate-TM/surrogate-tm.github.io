Ячейка LineWeight (раздел Line Format)

Определяет толщину линий фигуры. Чтобы задать толщину, укажите число с допустимой единицей измерения.

Заметки

Задать значение LineWeight можно также в диалоговом окне Линия (чтобы вызвать его, на вкладке Главная в группе Фигура щелкните элемент Линия, наведите указатель на пункт Толщина и выберите команду Другие линии).

Если единица измерения не указана, используется единица измерения, заданная для текста в диалоговом окне Параметры Visio (чтобы вызвать его, откройте вкладку Файл и щелкните элемент Параметры). Толщина линий не зависит от масштаба документа. При масштабировании документа толщина линий остается прежней.

Чтобы получить ссылку на ячейку LineWeight по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

LineWeight

Чтобы получить ссылку на ячейку LineWeight по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowLine

Индекс ячейки:

visLineWeight
