Ячейка ShapeFixedCode (раздел "Shape Layout")

Указывает поведение размещаемой фигуры при ее размещении.

Значение

Режим выбора

Константа автоматизации

&H1

Не перемещать фигуру, если фигуры были размещены с помощью диалогового окна Настройка макета.

visSLOFixedPlacement

&H2

Не перемещать эту фигуру и не разрешать размещение поверх нее сдвигаемых фигур.

visSLOFixedPlow

&H4

Не перемещать эту фигуру и разрешить размещение поверх нее сдвигаемых фигур.

visSLOFixedPermeablePlow

&H20 (32)

Игнорировать расположение точек соединения, к которым ведет маршрут.

visSLOFixedConnPtsIgnore

&H40 (64)

Разрешить маршруты только к сторонам с точками соединения.

visSLOFixedConnPtsOnly

&H80 (128)

Не приклеивать к периметру этой фигуры. Вместо этого приклеивать к рамке выравнивания фигуры.

visSLOFixedNoFoldToShape

Заметки

Это значение можно также задать на вкладке Размещение диалогового окна Поведение (выберите фигуру, на вкладке Разработчик в группе Конструктор фигур выберите пункт Поведение и откройте вкладку Размещение).

Можно задать любое сочетание этих значений для ячейки. Например, можно ввести значение 3 (&H3), которое запрещает движение при размещении фигур с помощью диалогового окна Настройка макета (на вкладке Конструктор в группе Макет щелкните Изменить расположение фигур, а затем выберите пункт Дополнительные параметры разметки) и при помещении других размещаемых фигур на фигуру или рядом с ней.

В более ранних версиях, чем Visio 2000, для настройки этого поведения используется ячейка ObjInteract в разделе "Miscellaneous".

Чтобы создать ссылку на ячейку ShapeFixedCode по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapeFixedCode

Чтобы создать ссылку на ячейку ShapeFixedCode из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:

visSLOFixedCode
