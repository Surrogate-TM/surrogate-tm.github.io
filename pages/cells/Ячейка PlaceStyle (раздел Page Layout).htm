Заметки

Задать значение этой ячейки также можно в диалоговом окне Настройка макета.

Чтобы получить ссылку на ячейку PlaceStyle по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

PlaceStyle

Чтобы получить ссылку на ячейку PlaceStyle по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowPageLayout

Индекс ячейки:

visPLOPlaceStyle
