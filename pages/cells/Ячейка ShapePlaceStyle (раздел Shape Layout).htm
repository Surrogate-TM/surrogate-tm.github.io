Ячейка ShapePlaceStyle (раздел "Shape Layout")

Указывает способ расположения фигур на странице при использовании диалогового окна Настройка макета (на вкладке Конструктор в группе Макет щелкните Изменить расположение фигур, а затем выберите пункт Дополнительные параметры разметки). Хранит значения стиля макета и выравнивания из VisCellIndices.

Заметки

Чтобы создать ссылку на ячейку ShapePlaceStyle по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapePlaceStyle

Чтобы создать ссылку на ячейку ShapePlaceStyle из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:

visSLOPlaceStyle
