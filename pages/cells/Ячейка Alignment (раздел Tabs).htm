Ячейка Alignment (раздел Tabs)

Указывает выравнивание при табуляции.

Значение

Выравнивание

Константа автоматизации

0

По левому краю

visTabStopLeft

1

По центру

visTabStopCenter

2

По правому краю

visTabStopRight

3

Точка

visTabStopDecimal

4

Запятая

visTabStopComma

Примечания

Чтобы получить ссылку на ячейку Alignment по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Tabs.ij,
где i и j = <1>, 2, 3

Чтобы получить ссылку на ячейку выравнивания Alignment по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionTab

Индекс строки:

visRowTab + i,
где i = 0, 1, 2...

Индекс ячейки:

(j*3) + visTabAlign
