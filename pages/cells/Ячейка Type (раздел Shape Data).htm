Ячейка Type (раздел "Shape Data")

Задает тип данных для значения данных фигуры.

Значение

Описание

Константа автоматизации

0

Строка. Это значение задано по умолчанию.

visPropTypeString

1

Постоянный список. Отображает элементы списка в виде поля с раскрывающимся списком в диалоговом окне Определить данные фигуры. Элементы списка необходимо указать в ячейке Format. Пользователи могут выбрать только один элемент из списка.

visPropTypeListFix

2

Число. Включает значения даты, времени, длительности, денежные значения, а также скалярные величины, измерения и углы. Представление формата можно задать в ячейке Format.

visPropTypeNumber

3

Логическое значение. Отображает значения FALSE и TRUE в виде элементов, которые пользователи могут выбрать в раскрывающемся списке в диалоговом окне Определить данные фигуры.

visPropTypeBool

4

Переменный список. Отображает элементы списка в виде поля с раскрывающимся списком в диалоговом окне Определить данные фигуры. Элементы списка необходимо указать в ячейке Format. Пользователи могут выбрать элемент списка или ввести новый элемент, который будет добавлен к текущему списку в ячейке Format.

visPropTypeListVar

5

Значение даты или времени. Отображает дни, месяцы и годы или секунды, минуты и дни, а также объединенные значения даты и времени. Представление формата можно задать в ячейке Format.

visPropTypeDate

6

Значение длительности. Отображает затраченное время. Представление формата можно задать в ячейке Format.

visPropTypeDuration

7

Денежное значение, основанное на текущих региональных параметрах системы. Представление формата можно задать в ячейке Format.

visPropTypeCurrency

Заметки

Чтобы создать ссылку на ячейку Type по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Prop.имя.Type,

где Prop.имя — это имя строки

Чтобы создать ссылку на ячейку Type из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionProp

Индекс строки:

visRowProp + i,

где i = 0, 1, 2...

Индекс ячейки:

visCustPropsType
