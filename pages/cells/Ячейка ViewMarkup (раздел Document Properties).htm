Ячейка ViewMarkup (раздел "Document Properties")

Определяет, отображаются ли исправления в окне документа.

Значение

Описание

TRUE

Исправления в документе отображаются.

FALSE

Исправления не отображаются (значение по умолчанию).

Заметки

Если включено отслеживание исправлений (ячейка AddMarkup имеет значение TRUE), ячейке ViewMarkup автоматически присваивается значение TRUE, которое сохраняется даже после отключения отслеживания исправлений (когда значение ячейки AddMarkup меняется на FALSE). Если ячейка AddMarkup имеет значение TRUE, значение ячейки ViewMarkup игнорируется.

Ячейке ViewMarkup также присваивается значение TRUE при вводе примечаний в документ (даже если отслеживание исправлений отключено). Для просмотра примечаний в документе эта ячейка должна иметь значение TRUE.

Эта ячейка соответствует команде Показывать слои исправлений в группе Исправления на вкладке Проверка.

Чтобы создать ссылку на ячейку ViewMarkup по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ViewMarkup

Чтобы создать ссылку на ячейку ViewMarkup из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowDoc

Индекс ячейки:

visDocViewMarkup
