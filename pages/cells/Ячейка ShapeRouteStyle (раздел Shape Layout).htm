Заметки

Значение этой ячейки для отдельного соединителя можно также задать на вкладке Соединительная линия диалогового окна Поведение (выберите соединитель, на вкладке Разработчик выберите пункт Поведение и откройте вкладку Соединительная линия).

Чтобы задать это поведение для всех соединителей на странице, используйте ячейку RouteStyle в разделе "Page Layout".

В более ранних версиях, чем Visio 2000, для настройки этого поведения используется ячейка ObjBehavior в разделе "Miscellaneous".

Чтобы создать ссылку на ячейку ShapeRouteStyle по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapeRouteStyle

Чтобы создать ссылку на ячейку ShapeRouteStyle из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:

visSLORouteStyle
