Ячейка YGridSpacing (раздел "Ruler & Grid")

Указывает расстояние между вертикальными линиями фиксированной сетки (YGridDensity = 0).

Заметки

Эта ячейка соответствует параметру Минимальный интервал по вертикали в диалоговом окне Линейка и сетка (на вкладке Вид щелкните стрелку Показать).

Чтобы создать ссылку на ячейку YGridSpacing по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

YGridSpacing

Чтобы создать ссылку на ячейку YGridSpacing из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowRulerGrid

Индекс ячейки:

visYGridSpacing
