Заметки

Чтобы задать значение этой ячейки, можно также выбрать динамическую соединительную линию, открыть вкладку Разработчик, щелкнуть в группе Конструктор фигур элемент Поведение, а затем открыть вкладку Соединительная линия.

Чтобы получить ссылку на ячейку ConLineJumpStyle по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.
