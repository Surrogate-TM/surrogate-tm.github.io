Ячейка DoubleULine (раздел Character)

Определяет, применено ли к фрагменту текста подчеркивание двойной чертой.

Значение

Описание

TRUE

Текст подчеркнут двойной чертой.

FALSE

Текст не подчеркнут двойной чертой.

Заметки

Ячейка DoubleULine содержит сведения о форматировании, примененном к фрагменту текста в фигуре, если раздел Character состоит из нескольких строк. В противном случае она содержит сведения о форматировании всего текста в фигуре.

Задать значение этой ячейки также можно с помощью диалогового окна Текст (чтобы его вызвать, щелкните стрелку Шрифт на вкладке Главная).

Чтобы получить ссылку на ячейку DoubleULine по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Char.DblUnderline[i],
где i = <1>, 2, 3...

Чтобы получить ссылку на ячейку DoubleULine по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionCharacter

Индекс строки:

visRowCharacter + i,
где i = 0, 1, 2...

Индекс ячейки:

visCharacterDblUnderline
