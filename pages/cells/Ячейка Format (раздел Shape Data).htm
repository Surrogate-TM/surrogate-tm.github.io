Ячейка Format (раздел Shape Data)

Определяет форматирование элемента данных фигуры (строка, постоянный список, число, переменный список, дата или время, продолжительность или денежная единица).

Примечания

Тип элемента данных фигуры

Значение

Содержимое ячейки Format

Строка

0

Представление формата, соответствующее этому типу данных.

Постоянный список

1

Элементы, из которых будут состоять список, разделенные точками с запятой.

Число

2

Представление формата, соответствующее этому типу данных.

Переменный список

4

Элементы, из которых будут состоять список, разделенные точками с запятой.

Дата или время

5

Представление формата, соответствующее этому типу данных.

Продолжительность

6

Представление формата, соответствующее этому типу данных.

Денежная единица

7

Представление формата, соответствующее этому типу данных.

Примером представления формата, соответствующего типу данных, может служить формат "# #/4 UU", который форматирует число "12,43 in." как "12 2/4 INCHES". Дополнительные сведения о том, как задается представление формата, см. в разделе О представлениях формата.

Примером списка, состоящего из нескольких элементов, является строка "Проектирование;Кадры;Продажи;Маркетинг".

Значения дат (тип 5) отображаются в кратком формате даты. Значения денежных единиц (тип 7) отображаются с учетом текущего значения параметра Денежная единица на вкладке Региональные параметры компонента Язык и региональные стандарты панели управления.

Число (тип 2) может представлять размер, скалярное значение, угол, дату, время или денежную единицу. Чтобы гарантировать, что введенное число всегда будет интерпретироваться как дата, время или денежная единица, используйте вместо представления формата в ячейке Format функцию DATETIME или CY.

Чтобы получить ссылку на ячейку Format по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Prop.имя.Format,
где Prop.имя — имя строки

Чтобы получить ссылку на ячейку Format по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionProp

Индекс строки:

visRowProp + i,
где i = 0, 1, 2...

Индекс ячейки:

visCustPropsFormat
