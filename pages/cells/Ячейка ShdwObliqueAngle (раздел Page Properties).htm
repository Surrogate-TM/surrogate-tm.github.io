Ячейка ShdwObliqueAngle (раздел "Page Properties")

Содержит число, которое задает угол наклона при использовании типа тени по умолчанию.

Замечания

Значение "нуль" (0) для этой ячейки соответствует прямому углу. Значение угла увеличивается по часовой стрелке.

Угол, указанный в этой ячейке, используется, если для ячейки ShapeShdwType (тип тени для фигуры на странице) установлено значение "Использовать параметры страницы, заданные по умолчанию" (visFSTPageDefault), а тень наклонная. Тип тени, используемый по умолчанию на странице, определяется в ячейке ShdwType.

Чтобы настроить это поведение для отдельной фигуры, используйте ячейку ShapeShdwObliqueAngle в разделе "Fill Format".

Чтобы создать ссылку на ячейку ShdwObliqueAngle по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShdwObliqueAngle

Чтобы создать ссылку на ячейку ShdwObliqueAngle из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowPage

Индекс ячейки:

visPageShdwObliqueAngle
