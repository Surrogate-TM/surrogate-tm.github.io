Ячейка ShapeShdwType (раздел "Fill Format")

Определяет тип тени для фигуры.

Значение

Описание

Константа автоматизации

0

Использовать параметры страницы, заданные по умолчанию (по умолчанию)

visFSTPageDefault

1

Простая

visFSTSimple

2

Наклонная

visFSTOblique

Заметки

Эта ячейка позволяет применить к фигуре тень, которая отличается от используемой на странице по умолчанию (тип тени по умолчанию для страницы определяется в ячейке ShdwType раздела "Page Properties").

Типы простых теней называются в интерфейсе тенями со смещением. Простая тень создает впечатление тени фигуры, спроецированной на параллельную плоскость, расположенную за ней. Наклонные тени имеют в интерфейсе такое же название и создают впечатление тени, спроецированной на плоскость, перпендикулярную фигуре.

Список готовых типов простых и наклонных теней см. в поле Стиль диалогового окна Тень (на вкладке Главная в группе Фигура выберите пункт Тень и командуПараметры тени).

Чтобы создать ссылку на ячейку ShapeShdwType по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapeShdwType

Чтобы создать ссылку на ячейку ShapeShdwType из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

Индекс ячейки:
