Ячейка PageScale (раздел Page Properties)

Задает значение единицы страницы в текущем масштабе документа. Масштаб документа для страницы — это отношение единицы страницы в ячейке PageScale к единице документа в ячейке DrawingScale.

Заметки

Значение ячейки PageScale также можно задать на вкладке Масштаб документа в диалоговом окне Параметры страницы (на вкладке Конструктор щелкните стрелку Параметры страницы). Значение данной ячейки — это первое из двух чисел в поле Заранее определенный масштаб или Настраиваемый масштаб в зависимости от параметра масштаба документа, выбранного на вкладке Масштаб документа. Например, если для документа выбран архитектурный масштаб, масштаб документа для страницы составляет 3/32" = 1'0". Значение в ячейке PageScale будет составлять 0,0938 дюйма (или 3/32"), а значение в ячейке DrawingScale — 1 фут.

Чтобы получить ссылку на ячейку PageScale по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

PageScale

Чтобы получить ссылку на ячейку PageScale по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowPage

Индекс ячейки:

visPageScale
