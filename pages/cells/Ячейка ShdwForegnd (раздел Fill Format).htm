Ячейка ShdwForegnd (раздел "Fill Format")

Определяет цвет, используемый для переднего плана (штриха) узора заливки тени фигуры.

Заметки

Чтобы задать цвет, введите число от 0 до 23 (индекс из коллекции цветов).

Чтобы ввести пользовательский цвет, используйте функцию RGB или HSL. Значением пользовательского цвета является цвет RGB, и в окне таблицы свойств фигуры будет отображаться RGB(r, g, b), а не число. При использовании в числовых операциях пользовательским цветам назначаются значения 24 и выше.

Прозрачность цвета переднего плана для узора заливки тени фигуры можно задать с помощью ячейки ShdwForegndTrans.

Чтобы создать ссылку на ячейку ShdwForegnd по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShdwForegnd

Чтобы создать ссылку на ячейку ShdwForegnd из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowFill

Индекс ячейки:

visFillShdwForegnd
