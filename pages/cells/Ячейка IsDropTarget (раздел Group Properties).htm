Ячейка IsDropTarget (раздел Group Properties)

Определяет, поддерживает ли группа добавление фигур путем перетаскивания.

Значение

Описание

TRUE

Фигуру можно добавить в группу путем перетаскивания.

FALSE

Фигуру нельзя перетащить в группу.

Заметки

Чтобы задать это значение, можно также выбрать группу, открыть вкладку Разработчик, щелкнуть элемент Поведение и установить флажок Принять размещенные фигуры.

Чтобы добавить фигуру в группу путем перетаскивания, необходимо также включить эту возможность для фигуры. Для этого выберите фигуру, откройте вкладку Разработчик, щелкните элемент Поведение и установите флажок Добавить фигуру в группы при размещении. Это значение хранится в ячейке IsDropSource в разделе Miscellaneous.

Чтобы получить ссылку на ячейку IsDropTarget по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

IsDropTarget
