Ячейка ShapePermeablePlace (раздел "Shape Layout")

Определяет, можно ли помещать размещаемые фигуры поверх фигуры при размещении фигур с помощью с помощью диалогового окна Настройка макета (на вкладке Конструктор в группе Макет щелкните Изменить расположение фигур, а затем выберите пункт Дополнительные параметры разметки).

Значение

Описание

TRUE

Позволяет размещать поверх фигуры другие фигуры.

FALSE

Запрещает размещать поверх фигуры другие фигуры.

Заметки

Это значение можно также задать на вкладке Размещение диалогового окна Поведение (выберите фигуру, на вкладке Разработчик в группе Конструктор фигур выберите пункт Поведение и откройте вкладку Размещение).

В более ранних версиях, чем Visio 2000, для настройки этого поведения используется ячейка ObjInteract в разделе "Miscellaneous".

Чтобы создать ссылку на ячейку ShapePermeablePlace по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapePermeablePlace

Чтобы создать ссылку на ячейку ShapePermeablePlace из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:

visSLOPermeablePlace
