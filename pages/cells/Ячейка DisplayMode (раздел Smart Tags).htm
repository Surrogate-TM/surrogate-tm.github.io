Ячейка DisplayMode (раздел Smart Tags)

Определяет, отображается ли тег действия при наведении на него указателя мыши, при выборе фигуры либо постоянно.

  Примечание.

В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами.

Значение

Режим отображения

Константа автоматизации

0

Тег отображается, когда указатель мыши останавливается над тегом (режим по умолчанию).

visSmartTagDispModeMouseOver

1

Тег отображается, когда фигура выбрана.

visSmartTagDispModeShapeSelected

2

Тег отображается все время.

visSmartTagDispModeAlways

Примечания

Теги действий не отображаются в опубликованных данных и в данных, выведенных на печать.

Если для страницы задан тег действия и эта ячейка содержит значение 1, тег не отображается никогда, поскольку выбрать страницу невозможно.

Чтобы получить ссылку на ячейку DisplayMode по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

SmartTags.имя.DisplayMode,
где SmartTags.имя — имя строки тега действия

Чтобы получить ссылку на ячейку DisplayMode по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionSmartTag

Индекс строки:

visRowSmartTag + i,
где i = 0, 1, 2...

Индекс ячейки:

visSmartTagDisplayMode
