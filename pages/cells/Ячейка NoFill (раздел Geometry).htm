Ячейка NoFill (раздел Geometry)

Указывает, можно ли выполнить заливку пути.

Значение

Описание

TRUE

Заливка пути не выполняется, даже если выполнена заливка других путей в данной фигуре.

FALSE

К пути применяется заливка фигуры, даже если он не замкнут.

Примечания

Если для фигуры выбрана заливка без узора (значение 0), ее пути не заполняются. Эта ячейка позволяет выборочно отключать заливку для пути в фигуре.

Чтобы получить ссылку на ячейку NoFill по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Geometryi.NoFill,
где i = <1>, 2, 3...

Чтобы получить ссылку на ячейку NoFill по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionFirstComponent + i,
где i = 0, 1, 2...

Индекс строки:

visRowComponent

Индекс ячейки:

visCompNoFill
