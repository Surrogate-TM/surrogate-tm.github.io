Заметки

Чтобы задать размер документа, воспользуйтесь диалоговым окном Параметры документа (чтобы его вызвать, щелкните стрелку Параметры документа на вкладке Конструктор) или измените размер страницы вручную с помощью мыши.

Чтобы получить ссылку на ячейку DrawingSizeType по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

DrawingSizeType

Чтобы получить ссылку на ячейку DrawingSizeType по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowPage

Индекс ячейки:

visPageDrawSizeType
