Ячейка XRulerOrigin (раздел "Ruler & Grid")

Указывает точку начала координат на линейке по оси x для страницы.

Заметки

Эта ячейка соответствует параметру Ноль на горизонтальной линейке в диалоговом окне Линейка и сетка (на вкладке Вид щелкните стрелку Показать).

Чтобы создать ссылку на ячейку XRulerOrigin по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

XRulerOrigin

Чтобы создать ссылку на ячейку XRulerOrigin из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowRulerGrid

Индекс ячейки:

visXRulerOrigin
