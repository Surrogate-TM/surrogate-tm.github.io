Ячейка IsDropSource (раздел Miscellaneous)

Определяет, можно ли добавить фигуру в группу путем перетаскивания.

Значение

Описание

TRUE

Фигуру можно добавить в группу путем перетаскивания.

FALSE

Фигуру нельзя добавить в группу.

Заметки

Чтобы задать это значение, можно также выбрать фигуру, открыть вкладку Разработчик, щелкнуть элемент Поведение и установить флажок Добавить фигуру в группы при размещении.

Кроме включения этой возможности для фигуры необходимо также разрешить для группы прием фигур, которые в нее перетаскиваются. Для этого выберите группу, откройте вкладку Разработчик, щелкните элемент Поведение и установите флажок Принять размещенные фигуры. Это значение хранится в ячейке IsDropTarget в разделе Group Properties.

Чтобы получить ссылку на ячейку IsDropSource по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

IsDropSource
