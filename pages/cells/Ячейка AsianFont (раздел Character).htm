Ячейка AsianFont (раздел Character)

Содержит номер шрифта, используемого для форматирования текста с символами азиатского языка. Номера шрифтов зависят от того, какие шрифты установлены в системе.

Заметки

Шрифты для азиатских языков указаны на вкладке Шрифт в диалоговом окне Текст (щелкните стрелку в группе Шрифт на вкладке Главная). Этот список отображается только в том случае, если с помощью диалогового окна Языковые параметры Microsoft Office был добавлен язык, содержащий символы азиатских языков или из наборов сложных знаков (чтобы его вызвать, нажмите кнопку Пуск и выберите по очереди пункты Все программы, Microsoft Office, Средства Microsoft Office и Языковые параметры Microsoft Office.

Число 0 означает, что шрифт не указан. Если латинский шрифт или шрифты по умолчанию содержат необходимые знаки, используются они.

Чтобы получить ссылку на ячейку AsianFont по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Char.AsianFont[i],
где i = <1>, 2, 3...

Чтобы получить ссылку на ячейку AsianFont по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionCharacter

Индекс строки:

visRowCharacter + i,
где i = 0, 1, 2...

Индекс ячейки:

visCharacterAsianFont
