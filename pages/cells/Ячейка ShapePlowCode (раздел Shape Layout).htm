Ячейка ShapePlowCode (раздел "Shape Layout")

Определяет, сдвигается ли данная размещаемая фигура при перетаскивании к ней другой такой фигуры на странице документа.

Значение

Описание

Константа автоматизации

0

Сдвигается в соответствии с параметрами страницы.

visSLOPlowDefault

1

Никакие фигуры не сдвигаются.

visSLOPlowNever

2

Все фигуры сдвигаются.

visSLOPlowAlways

Заметки

Это значение для отдельной ячейки можно также задать на вкладке Размещение диалогового окна Поведение (выберите фигуру, на вкладке Разработчик в группе Конструктор фигур выберите пункт Поведение и откройте вкладку Размещение).

Чтобы задать это поведение для всех фигур на странице документа, используйте ячейку PlowCode в разделе "Page Layout".

Чтобы создать ссылку на ячейку ShapePlowCode по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapePlowCode

Чтобы создать ссылку на ячейку ShapePlowCode из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:

visSLOPlowCode
