Ячейка LockCustProp (раздел Protection)

Определяет, может ли пользователь добавлять, удалять и изменять данные фигуры в пользовательском интерфейсе с помощью диалогового окна Определение данных фигуры или контекстного меню окна Данные фигуры.

Значение

Описание

TRUE

Команда Определить данные фигуры в контекстном меню окна Данные фигуры отключена.

FALSE

Команда Определить данные фигуры в контекстном меню окна Данные фигуры активна (вариант по умолчанию).

Заметки

Значение TRUE не запрещает пользователю изменять элемент данных фигуры или вносить изменения в разделе Shape Data окна таблицы свойств фигуры.

Чтобы получить ссылку на ячейку LockCustProp по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

LockCustProp

Чтобы получить ссылку на ячейку LockCustProp по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowLock

Индекс ячейки:

visLockCustProp
