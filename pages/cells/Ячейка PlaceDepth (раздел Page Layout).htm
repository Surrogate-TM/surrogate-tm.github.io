Ячейка PlaceDepth (раздел Page Layout)

Определяет способ анализа документа перед созданием макета и тип макета.

Значение

Глубина размещения для вертикальных и горизонтальных макетов

Константа автоматизации

0

Значение по умолчанию для страницы

visPLOPlaceDepthDefault

1

Средняя

visPLOPlaceDepthMedium

2

Глубокая

visPLOPlaceDepthDeep

3

Мелкая

visPLOPlaceDepthShallow

Примечания

Чтобы получить ссылку на ячейку PlaceDepth по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

PlaceDepth

Чтобы получить ссылку на ячейку PlaceDepth по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowPageLayout

Индекс ячейки:

visPLOPlaceDepth
