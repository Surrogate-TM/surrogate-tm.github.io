Ячейка ConLineJumpCode (раздел Shape Layout)

Определяет, как пересекаются соединительные линии.

Значение

Описание

Константа автоматизации

0

В соответствии с параметрами страницы. Чтобы увидеть параметры страницы, на вкладке Конструктор щелкните стрелку в группе Параметры страницы, а затем откройте вкладку Макеты и маршруты.

visSLOJumpDefault

1

Никогда

visSLOJumpNever

2

Всегда

visSLOJumpAlways

3

Переход выполняется для другой соединительной линии

visSLOJumpOther

4

Переход не выполняется ни для одной соединительной лини

visSLOJumpNeither

Заметки

Чтобы задать значение этой ячейки, можно также выбрать динамическую соединительную линию, открыть вкладку Разработчик, щелкнуть в группе Конструктор фигур элемент Поведение, а затем открыть вкладку Соединительная линия.

Чтобы получить ссылку на ячейку ConLineJumpCode по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

ConLineJumpCode

Чтобы получить ссылку на ячейку ConLineJumpCode по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowShapeLayout

Индекс ячейки:
