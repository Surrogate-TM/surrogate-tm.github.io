Ячейка Disabled (раздел Action Tags)

Указывает, отображается ли тег действия в окне документа.

  Примечание.

В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами.

Значение

Описание

TRUE

Тег действия отключен.

FALSE

Тег действия включен (вариант по умолчанию).

Примечания

Если тег действия отключен, он не отображается до тех пор, пока не будет активирован снова.

Чтобы получить ссылку на ячейку Disabled по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

SmartTags.имя.Disabled,
где SmartTags.имя — имя строки тегов действий

Чтобы получить ссылку на ячейку Disabled по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionSmartTag

Индекс строки:

visRowSmartTag + i,
где i = 0, 1, 2...

Индекс ячейки:

visSmartTagDisabled
