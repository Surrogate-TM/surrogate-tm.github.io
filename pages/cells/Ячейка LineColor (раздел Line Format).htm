Ячейка LineColor (раздел Line Format)

Определяет цвет линии фигуры.

Заметки

Чтобы задать цвет линии, введите число от 0 до 23, которое будет представлять индекс в наборе цветов линий. Просмотреть набор цветов линий можно в диалоговом окне Линия (на вкладке Главная в группе Фигура щелкните элемент Линия, наведите указатель на пункт Толщина и выберите команду Другие линии). Задать значение LineColor можно также в диалоговом окне Линия.

Для ввода пользовательского цвета воспользуйтесь функцией RGB или HSL. Значение пользовательского цвета представляется в системе RGB, и в окне таблицы свойств фигуры вместо числа отображается значение RGB(r, g, b). В операциях с числами пользовательские цвета принимают значения от 24 и выше.

Задать прозрачность цвета линии можно с помощью ячейки LineColorTrans.

Чтобы получить ссылку на ячейку LineColor по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

LineColor

Чтобы получить ссылку на ячейку LineColor по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowLine

Индекс ячейки:

visLineColor
