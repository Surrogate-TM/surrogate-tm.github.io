Ячейка Value (раздел "Shape Data")

Содержит значение элемента данных фигуры, введенное в диалоговом окне Определить данные фигуры.

Заметки

Формулы, введенные в этой ячейке, переопределяются значениями, указанными в диалоговом окне Определить данные фигуры. Это происходит даже при использовании функции GUARD для защиты формулы.

Чтобы создать ссылку на ячейку Value по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Prop.имя.Value,

где Prop.имя — это имя строки

Чтобы создать ссылку на ячейку Value из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionProp

Индекс строки:

visRowProp + i,

где i = 0, 1, 2...

Индекс ячейки:

visCustPropsValue
