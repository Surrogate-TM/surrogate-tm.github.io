Ячейка X (раздел "Controls")

Обозначает координату x, указывающую расположение управляющего маркера фигуры в локальной системе координат.

Примечания

Чтобы создать ссылку на ячейку X по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Controls.имя.X,

где Controls.имя — это имя строки элементов управления.

Чтобы создать ссылку на ячейку X из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionControls

Индекс строки:

visRowControl + i,
где i = 0, 1, 2...

Индекс ячейки:

visCtlX
