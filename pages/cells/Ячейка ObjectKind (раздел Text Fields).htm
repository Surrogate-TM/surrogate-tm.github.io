Ячейка ObjectKind (раздел Text Fields)

Указывает тип текстового поля.

Значение

Описание

Константа автоматизации

0

Стандартное

visTFOKStandard

1

Горизонтальное в вертикальном

visTFOKHorizontaInVertical

Примечания

Текстовые поля могут быть следующих типов:

         стандартные (поле было вставлено по категории поля);

         горизонтальные в вертикальном (поле является горизонтальным текстовым и располагается в вертикальном текстовом поле).

Чтобы получить ссылку на ячейку ObjectKind по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Fields.ObjectKind[i],
где i = <1>, 2, 3...

Чтобы получить ссылку на ячейку ObjectKind по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionTextField

Индекс строки:

visRowField + i
where i = 0, 1, 2...

Индекс ячейки:

visFieldObjectKind
