Ячейка DocLangID (раздел Document Properties)

Указывает язык документа по умолчанию.

Заметки

Язык документа определяется на основе языка, указанного в диалоговом окне Языковые параметры Microsoft Office (чтобы его вызвать, нажмите кнопку Пуск и выберите по очереди пункты Все программы, Microsoft Office, Средства Microsoft Office и Языковые параметры Microsoft Office).

Чтобы изменить язык документа, вместо непосредственного редактирования ячейки лучше изменить ее значение в диалоговом окне Свойства (чтобы его вызвать, откройте вкладку Файл и щелкните элемент Сведения, затем — Свойства документа, а затем — Дополнительные свойства) либо программным образом с помощью свойства Language объекта Document.

Имя ячейки:

DocLangID

Чтобы получить ссылку на ячейку DocLangID по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowDoc

Индекс ячейки:

visDocLangID
