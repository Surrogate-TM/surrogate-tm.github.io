Ячейка Address (раздел Hyperlinks)

Указывает URL-адрес, имя файла или путь UNC, по которому следует перейти.

Адрес можно задать как путь относительно основного пути, заданного для документа в поле База гиперссылок на вкладке Документ диалогового окна Свойства(откройте вкладку Файл и щелкните элемент Сведения, затем — Свойства документа, а затем — Дополнительные свойства). Если основной путь для документа не задан, приложение осуществляет переход, руководствуясь путем документа. Если документ не был сохранен, гиперссылка не определена.

Заметки

Значение ячейки Address также можно задать в диалоговом окне Гиперссылки (чтобы вызвать его, щелкните элемент Гиперссылки на вкладке Вставка).

Чтобы получить ссылку на ячейку Address по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Имя ячейки:

Hyperlink.имя.Address,
где Hyperlink.имя — имя строки гиперссылки

Чтобы получить ссылку на ячейку Address по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Индекс раздела:

visSectionHyperlink

Индекс строки:

visRow1stHyperlink + i,
где i = 0, 1, 2...

Индекс ячейки:

visHLinkAddress
