Ячейка PreviewScope (раздел Document Properties)

Определяет, содержит ли документ представление предварительного просмотра. Если да, эта ячейка определяет, отображается ли в представлении предварительного просмотра только первая страница или все страницы документа.

Значение

Область предварительного просмотра

Константа автоматизации

0

Первая страница

visDocPreviewScope1stPage

1

Нет

visDocPreviewScopeNone

2

Все страницы

visDocPreviewScopeAllPages

Заметки

Задать это значение также можно на вкладке Документ диалогового окна Свойства (нажмите кнопку Office, откройте вкладку Сведения и щелкните элемент Свойства документа, а затем — Дополнительные свойства).

Чтобы получить ссылку на ячейку PreviewScope по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

PreviewScope

Чтобы получить ссылку на ячейку PreviewScope по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObj

Индекс строки:

visRowDoc

Индекс ячейки:

visDocPreviewScope
