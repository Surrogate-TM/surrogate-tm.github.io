Ячейка OutputFormat (раздел Document Properties)

Определяет формат вывода документа. Страницы документа обычно форматируются для печати (вариант по умолчанию), однако можно выбрать и другой формат вывода.

Значение

Формат вывода

0

Печать (режим по умолчанию)

1

Показ слайдов PowerPoint

2

Формат HTML или GIF

Примечания

Чтобы получить ссылку на ячейку OutputFormat по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

OutputFormat

Чтобы получить ссылку на ячейку OutputFormat по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowDoc

Индекс ячейки:

visDocOutputFormat
