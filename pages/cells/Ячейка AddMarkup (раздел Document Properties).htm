Ячейка AddMarkup (раздел Document Properties)

Указывает, находится ли документ в режиме проверки исправлений.

Значение

Описание

TRUE

Документ находится в режиме проверки.

FALSE

Документ не находится в режиме проверки (режим по умолчанию).

Заметки

Если для ячейки AddMarkup задано значение TRUE, рецензент добавляет исправления и изменения применяются к страницам слоев исправлений, а не к страницам исходного документа. Если для ячейки AddMarkup задано значение FALSE, отслеживание исправлений отключено и изменения применяются к страницам исходного документа.

  Примечание.

Отключить внесение исправлений в документ можно с помощью функции GUARD. Если ячейка AddMarkup содержит формулу =GUARD(FALSE), команда Записывать исправления отключена.

Этот параметр соответствует параметру команды Записывать исправления в группе Исправления на вкладке Просмотр.

Чтобы получить ссылку на ячейку AddMarkup по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.
