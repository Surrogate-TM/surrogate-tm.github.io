Ячейка UIVisibility (раздел "Page Properties")

Определяет, доступно ли имя страницы в пользовательском интерфейсе.

Значение

Описание

Константа автоматизации

0

Имя страницы отображается в пользовательском интерфейсе (значение по умолчанию).

visUIVNormal

1

Имя страницы не отображается в пользовательском интерфейсе.

visUIVHidden

Заметки

Если ячейка UIVisibility имеет значение visUIVHidden, страница не будет указываться в тех разделах пользовательского интерфейса, где есть строка, содержащая имя страницы. Например, страница не будет доступна для выбора в проводнике по документам или на вкладках страниц. Тем не менее она доступна при использовании автоматизации или путей пользовательского интерфейса, которые не включают имя страницы, например при выборе команды Печать.

  Примечание.

Чтобы страница не указывалась в меню команды Печать документа, назначьте ее фоновой страницей (свойство Type имеет значение visTypeBackground), которая не используется в качестве фона другими страницами (фигуры на фоновой странице печатаются при печати связанной с ней страницы). Команда Печать документа поддерживает только индексированные страницы, а фоновые страницы не индексируются.

Эта ячейка предназначена для использования со страницами документа, а не со страницами слоев исправлений, для которых ячейка UIVisibility по умолчанию имеет значение visUIVHidden, которое нельзя изменить.

Чтобы создать ссылку на ячейку UIVisibility по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

UIVisibility

Чтобы создать ссылку на ячейку UIVisibility из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowPage

Индекс ячейки:

visPageUIVisibility
