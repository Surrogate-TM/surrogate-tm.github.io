Единицы измерения

При вставке полей в текст или построении формул часто требуется указать единицы измерения для вводимых значений.

Вычисление результата формулы зависит от того, в какую ячейку она введена. Обычно ячейки, определяющие положение фигуры, размер или угол, требуют ввода пары "единица — число измерения", которая состоит из числа и подходящей единицы измерения, которая необходима для обработки числа. Многие другие ячейки не требуют ввода единиц измерения и возвращают строку, значение TRUE или FALSE либо индекс. Например, формула, которая в ячейке FillForegnd обозначает цвет 5 из цветовой палитры документа, дает значение TRUE (и блокирует ширину ячейки) в ячейке LockWidth.

Всегда указывайте единицу измерения при вводе формулы в ячейку, которая требует число с единицей измерения. В противном случае используется единица измерения, заданная по умолчанию для этой ячейки, например одна из единиц измерения страницы, документа, типа, длительности или угла.

Дробные единицы измерения

Можно задать дробные единицы измерения в ячейке DrawingScale, чтобы изменить число промежуточных делений линейки, которые отображаются в окне документа. По умолчанию расстояния между основными делениями на линейках Visio делятся на десять частей. Если в ячейке DrawingScale заданы дробные единицы изменения, расстояния делятся следующим образом:

         на десять частей для visInchFrac и visMileFrac;

         на двенадцать частей для visFeetAndInches.

Дробные единицы измерения не влияют на другие ячейки, кроме DrawingScale.

Неявные единицы измерения

Для анализа и хранения пары "значение — число" могут использоваться явные и неявные единицы измерения. Если для числа явно заданы единицы измерения, оно всегда отображается с той единицей измерения, которая была изначально задана для него. Число с неявными единицами измерения всегда преобразуется в эквивалентное число в единицах измерения документа, страницы или угла (в соответствии с типом ячейки).

Например, предположим, что значение, равное 1 дюйму, введено в ячейку A с использованием явных единиц, а в ячейку B — с использованием неявных единиц. Для обеих ячеек используются единицы измерения документа. После этого в качестве единицы измерения, используемой по умолчанию на странице, задаются сантиметры. В ячейке A будет по-прежнему отображаться "1 дюйм", так как в ней заданы явные единицы измерения, которые не изменяются при изменении значений по умолчанию. В ячейке B теперь отображается "2,54 см" — эквивалентное значение в единицах измерения, заданных по умолчанию.

Чтобы указать единицы измерения неявно, используйте следующий синтаксис:

number [unit, flag]

number

Исходное значение, такое как 3,7, 1,7E-4 или 5 1/2.

unit

Единицы измерения, в которых первоначально выражено число number.

flag

Система единиц измерения, которая будет использоваться для отображения значения с неявной единицей измерения. Список возможных значений см. ниже.

Элемент flag должен быть равен одной из указанных ниже букв (в любом регистре), обозначающих систему единиц измерения, которая будет использоваться для отображения значения с неявной единицей измерения.

flag

Система единиц измерения

Пример

a, A

Единицы измерения угла

=5[град,A]

d, D

Единицы измерения документа

=5[дюйм,D]

e, E

Единицы измерения длительности

=5[час.,E]

p, P

Единицы измерения страницы

=5[дюйм,P]

t, T

Единицы измерения шрифта

=5[пт,T]

Кроме того, можно использовать значения DL, DP, DT, DA, DE для неявных единиц измерения документа, страницы, текста, угла и времени соответственно. Для таких единиц предполагается, что связанное значение выражено во внутренних единицах измерения. Например, если в текущей системе единиц измерения используются сантиметры, =2 DL интерпретируется как 2 внутренние единицы (2 дюйма) и отображается как "5,08 см".

Если использовать описанный выше синтаксис неявных единиц измерения, выражение (=2 DL) можно представить как 2[дюйм,d]. Синтаксис неявных единиц позволяет управлять тем, как интерпретируется значение, то есть можно также задать значение 2[фт,d], которое будет интерпретировано как 2 фута и отображено как "60,96 см". Неявные единицы измерения DL, DP, DT, DA и DE являются универсальными и не зависят от языка.
