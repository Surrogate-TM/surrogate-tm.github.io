Значения ошибок

Значения ошибок отображаются в ячейках с неправильными формулами.

Если формула ссылается на ячейку, которая содержит значение ошибки, для нее также отображается значение ошибки. Для поиска значений ошибки можно использовать функции ISERR, ISERRNA, ISERROR или ISERRVALUE.

Если в ячейке отображается

Формула содержит

Пример

#ДЕЛ/0!

Деление на 0

10/0

#ЗНАЧ!

Аргумент или операнд неверного типа

5 + "Дом"

#ССЫЛКА!

Ссылка на несуществующую ячейку

Ячейка, которая ссылается на удаленную ячейку

#ЧИСЛО!

Недопустимое число

Квадратный корень из отрицательного числа

#Н/Д!

Недоступное значение

Функция NA( )

#DIM!

Значение с единицей измерения, превышающее допустимый диапазон для единицы измерения (допустимыми степенями являются целые числа
-128 <= n <= 127)

Значение с единицей измерения, над которым выполняется недопустимая операция

1дюйм^100 * 1дюйм^100 (результатом является 1дюйм^200, что выходит за пределы допустимого диапазона для единицы измерения)

5,2см^1,5 (степень не является целым числом)
