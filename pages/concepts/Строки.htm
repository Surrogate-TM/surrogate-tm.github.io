Строки

Формулы могут содержать строки. Чтобы определить формат выводимой строки, например в ячейке с приглашением, значении элемента данных фигуры или текстовом поле, можно задать представление формата. Результат можно отформатировать как пару "число — единица измерения", строку, длительность или денежную единицу. Например, представление "0 #/10 uu" форматирует значение "10,9см" как "10 9/10 сантиметры".

Представление формата можно задать в ячейке Format в разделе Shape Data либо в виде аргумента функции FORMAT или FORMATEX. При вставке текстового поля представления формата указываются в списке форматов в диалоговом окне Поле (на вкладке Вставка).

Форматирование строк с помощью функций

В любой формуле, возвращающей строку, включая пользовательские формулы текстовых полей, можно использовать функцию FORMAT или FORMATEX. Функция FORMAT возвращает отформатированную строку. Функция FORMATEX преобразует нетипизированные входные данные в единицы измерения, заданные для форматирования результата.

Отображение отформатированных данных фигуры

Для форматирования отображаемого значения элемента данных фигуры можно ввести представление формата в ячейку Format.

Например, у фигуры временной шкалы может быть элемент данных, который измеряет стоимость процесса. По умолчанию такой элемент является строкой. Чтобы отформатировать строку "1200" в виде денежной единицы, можно ввести "$###,###.00" в ячейку Format.

При этом используется обозначение денежной единицы и разделитель разрядов, заданные на вкладке Денежные единицы диалогового окна Настройка формата в окне Язык и региональные стандарты на панели управления. (На панели управления выберите пункты Язык и региональные стандарты и Дополнительные параметры.)

Чтобы преобразовать строку в денежное значение для выполнения вычислений, используйте функцию CY.

Работа со строками с помощью функций

Функции можно использовать для работы с текстовыми строками, например для поиска или замены определенных символов в строке. Кроме того, можно получить сведения о позиции знака в строке, а также определить начальные и конечные символы текстовой строки.
