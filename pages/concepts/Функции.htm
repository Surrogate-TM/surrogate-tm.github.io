Функции

Функция выполняет одну четко определенную задачу. Большинство функций принимают в качестве входных значений несколько аргументов. Хотя число и тип аргументов зависит от функции, общий синтаксис всех функций одинаков:

FUNCTION(аргумент1, аргумент2, ...аргументN [, аргументA | аргументB])

Элемент синтаксиса

Описание

( )

Если у функции нет аргументов, за ней должны следовать пустые парные скобки ( ).

,

Аргументы разделяются запятой.

...

Используется только для обозначения в описании; не включайте этот знак в функцию.

[ ]

Необязательный аргумент. Используется только для обозначения в описании; не включайте этот знак в функцию.

|

Выбор; можно использовать аргументA или аргументB. Используется только для обозначения в описании; не включайте этот знак в функцию.

Многие функции, используемые в формулах, аналогичны функциям из электронных таблиц: математические, например SUM или SQRT; тригонометрические, например SIN или COS; логические, например IF или NOT. Многие другие функции, такие как GUARD, GRAVITY и RUNADDON, доступны только в Microsoft Office Visio.

Дополнительные сведения о конкретных функциях см. в этом справочнике по таблице свойств фигуры.

  Примечание.

Некоторые функции отображаются в формулах, созданных Visio, но не указываются в диалоговом окне Вставка функции и не описываются в этом справочнике, так как они зарезервированы для внутреннего использования и не должны применяться в других формулах. Примеры таких функций: ELLIPSE_THETA, _ELLIPSE_ECC, _UCON_C1 и _SHAPEMIN.
