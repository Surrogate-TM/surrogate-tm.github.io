Функция TEXTWIDTH

Возвращает ширину текста в фигуре.

Синтаксис

TEXTWIDTH(shapename!TheText[,maximumwidth])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

shapename!theText

Требуется

Строка

Ссылка на ячейку TheText в целевой фигуре; shapename! — имя фигуры, из которой требуется извлечь текст.

maximumwidth

Необязательно

Числовой

Максимальная ширина блока текста.

Возвращаемое значение
Строка

Заметки

Функция TEXTWIDTH обычно используется для подгонки ширины фигуры по размеру содержащегося в ней текста.

Если аргумент sheetN! опущен, по умолчанию используется текущая фигура.

Если задан аргумент maximumwidth, результатом будет строка максимальной длины, не превышающей этого значения. Если параметр maximumwidth опущен, возвращается ширина всего текста.

Пример

TEXTWIDTH(TheText)

Возвращает общую длину текста в текущей фигуре.
