Функция ISERROR

Возвращает значение ИСТИНА, если аргумент cellreference является ошибкой любого типа; в противном случае возвращает значение ЛОЖЬ. Функция ISERROR используется в формулах, ссылающихся на другие ячейки.

Синтаксис

ISERROR(cellreference)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

cellreference

Требуется

Строка

Ссылка на ячейку.

Пример 1

Ячейка

Формула

Возвращаемое значение

Scratch.A1

=NA( )

#Н/Д!

Scratch.B1

=ISERROR(Scratch.A1)

ИСТИНА

Возвращает значение ИСТИНА, поскольку ошибка #Н/Д! распознается функцией ISERROR. Распознать ошибку любого типа, кроме ошибки #Н/Д!, позволяет функция ISERR.

Пример 2

Ячейка

Формула

Возвращаемое значение

Scratch.X1

="Дом"

#ЗНАЧ!

Scratch.B1

=ISERROR(Scratch.X1)

ИСТИНА

Возвращает значение ИСТИНА, поскольку ошибка #ЗНАЧ! распознается функцией ISERROR. Чтобы построить выражение на основе значения ошибки #ЗНАЧ!, используйте функцию ISERRVALUE.
