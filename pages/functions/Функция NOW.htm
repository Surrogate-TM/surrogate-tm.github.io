Функция NOW

Возвращает текущую дату и время.

Синтаксис

NOW ( )

Возвращаемое значение
Дата и время

Заметки

Результат функции NOW автоматически пересчитывается каждую минуту.

Пример 1

NOW( )

Возвращает текущую дату и время, например "27.09.2010 12:03:30".

Пример 2

FORMAT(NOW();"дд ммм. гггг чч:мм")

Возвращает текущую дату и время в виде "27 сен. 2010 12:03".

Пример 3

NOW()+2НЕД.

Возвращает текущую дату и время с добавлением двух недель, например "11.10.2010 12:03:30".
