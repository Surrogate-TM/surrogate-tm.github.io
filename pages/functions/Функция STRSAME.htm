Функция STRSAME

Проверяет тождественность строк. Возвращает значение ИСТИНА, если строки тождественны, и ЛОЖЬ, если нет.

Синтаксис

STRSAME ("string1", "string2", ignoreCase)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

string1

Требуется

Строка

Первая строка для сравнения.

string2

Требуется

Строка

Вторая строка для сравнения.

ignoreCase

Необязательно

Логический

Значение ИСТИНА предписывает не учитывать регистр при сравнении, ЛОЖЬ — выполнять сравнение с учетом регистра. Значение по умолчанию — ЛОЖЬ.

Возвращаемое значение
Логическое

Заметки
Чтобы сравнить многобайтовые строки или выполнить сравнение строк с учетом правил использования регистра символов для определенного языкового стандарта, используйте функцию STRSAMEEX.

Пример 1

STRSAME("кот";"пес")

Возвращает значение ЛОЖЬ.

Пример 2

STRSAME("кот";"кот")

Возвращает значение ИСТИНА.

Пример 3

STRSAME("кот";"кот"; ИСТИНА)

Возвращает значение ИСТИНА.

Пример 4

STRSAME("кот";"КОТ")

Возвращает значение ЛОЖЬ.

Пример 5

STRSAME("кот";"КОТ"; ИСТИНА)

Возвращает значение ИСТИНА.

Пример 6

STRSAME("пёс;"ПЕС"; ИСТИНА)

Возвращает значение ЛОЖЬ.

Пример 7

STRSAME("пёс";"ПЁС"; ИСТИНА)

Возвращает значение ИСТИНА.
