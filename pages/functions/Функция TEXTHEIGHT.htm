Функция TEXTHEIGHT

Возвращает высоту текста в фигуре, длина строк которого не превышает значения maximumwidth.

Синтаксис

TEXTHEIGHT(shapename!TheText[,maximumwidth])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

shapename!theText

Требуется

Строка

Ссылка на ячейку TheText в целевой фигуре; shapename! — имя фигуры, из которой требуется извлечь текст.

maximumwidth

Необязательно

Числовой

Максимальная ширина блока текста.

Возвращаемое значение
Строка

Заметки
Возвращаемое значение представляет собой высоту текста с учетом пробелов до и после него, междустрочного интервала, а также верхнего и нижнего полей блока текста. Эта функция обычно используется для подгонки высоты фигуры по размеру содержащегося в ней текста.

Пример

TEXTHEIGHT(TheText;(Width - 0,5 см))

Возвращает высоту текста, разбитого на строки по ширине фигуры за вычетом 0,5 см.
