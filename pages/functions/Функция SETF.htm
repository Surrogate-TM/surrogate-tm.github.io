Функция SETF

Задает формулу ячейки.

Синтаксис

SETF( GETREF(cell), formula)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

cell

Требуется

Строка

Ячейка, для которой требуется задать формулу.

formula

Требуется

Строка

Задаваемая формула.

Заметки

При вычислении результат выражения в аргументе formula становится новой формулой в ячейке cell. Если аргумент formula заключен в кавычки, выражение в кавычках записывается в ячейку cell. Чтобы задать для этой ячейки строковое значение, заключите аргумент formula в три пары кавычек.

Чтобы избежать циклических ссылок, конечную ячейку нужно указать с помощью ссылки GETREF() или в виде строки. Рекомендуется использовать функцию GETREF, поскольку Microsoft Visio может скорректировать ссылки при переносе фигуры в другой документ.

Если аргумент cell задан не с помощью функции GETREF и не в виде строки, функция возвращает ошибку, а формула в ячейке не меняется. Если аргумент formula содержит синтаксическую ошибку, функция возвращает ошибку, а формула в ячейке cell не меняется.

Пример 1

SETF( GETREF(Scratch.A1); 1,5 см * 6 + 1 м)

Задает в качестве формулы в ячейке Scratch.A1 значение 109 см.

Пример 2

SETF( GETREF(Scratch.A1); "1,5 см * 6 + 1 м")

Задает в качестве формулы в ячейке Scratch.A1 выражение "1,5 см * 6 + 1 м".

Пример 3

SETF( GETREF(Scratch.A1); """Сказать """"ааа""""""")

Задает в качестве формулы в ячейке Scratch.A1 строку "Сказать ""ааа""", которая возвращает значение "Сказать "ааа"".
