Функция IFERROR

Возвращает вычисленный результат первичного выражения, если он не выдает ошибку. В противном случае возвращает вычисленный результат альтернативного выражения.

Синтаксис

IFERROR(primary expression, alternate expression)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

primary expression

Требуется

Строка

Первичное выражение, значение которого требуется определить.

alternate expression

Требуется

Строка

Альтернативное выражение для вычисления, если первичное выражение вычисляет ошибку.

Возвращаемое значение
Изменяется
