Функция FLOOR

Округляет число в сторону нуля до ближайшего целого или числа, кратного значению multiple.

Синтаксис

FLOOR(number, multiple)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

number

Требуется

Число

Округляемое значение.

multiple

Требуется

Число

Множитель, до которого округляется число.

Возвращаемое значение
Число

Заметки

Если аргумент multiple не определен, число округляется к нулю до ближайшего целого числа.

Аргументы number и multiple должны иметь одинаковый знак, иначе возвращается ошибка #ЧИСЛО!. Если аргумент number или multiple не удается преобразовать в значение, возвращается ошибка #ЗНАЧ!. Если значение number или multiple равно 0, результатом является 0.

Пример 1

FLOOR(3,7)

Возвращает 3.

Пример 2

FLOOR(-3,7)

Возвращает -3.

Пример 3

FLOOR(3,7; 0,5)

Возвращает 3,5.
