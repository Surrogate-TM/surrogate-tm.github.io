Функция AND

Возвращает значение ИСТИНА (1), если все предоставленные логические выражения имеют значение ИСТИНА. Если хотя бы одно из логических выражений имеет значение ЛОЖЬ или 0, функция И возвращает значение ЛОЖЬ (0).

Синтаксис

AND(logical expression1,logical expression2,...,logical expressionN)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

logical expression

Требуется

Строка

Сочетание констант, операторов, функций и ссылок на ячейки ShapeSheet, результатом которых является значение. Считается, что любое выражение, дающее ненулевое значение, имеет значение ИСТИНА.

Пример

AND(Height > 1, PinX > 1)

Возвращает ИСТИНА, если оба выражения имеют значение ИСТИНА. Возвращает ЛОЖЬ, если любое из выражений имеет значение ЛОЖЬ.
