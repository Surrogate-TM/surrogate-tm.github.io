Функция MINUTE

Возвращает целое число от 0 до 59, представляющее минуты в аргументе datetime или expression.

Синтаксис

MINUTE("datetime"|expression[,lcid])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

datetime

Требуется

Строка

Любая строка с датой и временем в общепринятом формате или ссылка на ячейку, содержащую дату и время.

expression

Требуется

Строка

Любое выражение, возвращающее дату и время.

lcid

Необязательно

Число

Код языка для использования при анализе значения даты и времени в формате другого региона. Код языка — это число, описанное в системных заголовочных файлах.

Возвращаемое значение
Целое число

Заметки

Компонент даты в аргументах datetime и expression отбрасывается.

Округление не выполняется. Если значение datetime отсутствует или не может быть преобразовано в допустимый результат, функция возвращает ошибку.

Возвращаемое значение форматируется в соответствии со стилем времени, заданным в текущих региональных параметрах системы.

Функция MINUTE также принимает в качестве параметра expression отдельное числовое значение, десятичная часть которого представляет долю суток, истекшую с полуночи.

Пример 1

MINUTE("7.07.1999 13:45:24")

Возвращает 45.

Пример 2

MINUTE(TIMEVALUE("25 января 1999 12:07:45")+6 мин)

Возвращает 13.

Пример 3

MINUTE(0,575)

Возвращает 48.
