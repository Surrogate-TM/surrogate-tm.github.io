Функция MID

Возвращает заданное количество символов из текстовой строки, начиная с указанной позиции.

Синтаксис

MID (text, start_num, num_chars)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

text

Требуется

Строка

Текстовая строка, содержащая символы, которые требуется извлечь.

start_num

Требуется

Число

Позиция первого символа, который требуется извлечь. Позиция первого символа текстовой строки — 1.

num_chars

Требуется

Число

Число символов, которые требуется получить.

Возвращаемое значение
Строка

Заметки

Если значение start_num:

         превышает длину строки text, функция MID возвращает значение "" (пустой текст);

         меньше длины строки text, но сумма аргументов start_num и num_chars превышает длину строки text, функция MID возвращает знаки до конца этой строки;

         меньше 1, функция MID возвращает значение ошибки #ЗНАЧ!.

Если значение аргумента num_chars отрицательно, функция MID возвращает значение ошибки #ЗНАЧ!.

Пример

MID ("ССН 999-99-9999";5;11)

Возвращает 999-99-9999.
