Функция QUEUEMARKEREVENT

Предписывает приложению запустить событие маркера для дополнительного компонента, кода Microsoft Visual Basic для приложений (VBA) или надстройки COM.

Синтаксис

QUEUEMARKEREVENT (event_string )

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

event_string

Требуется

Строка

Строка, которая передается обработчику событий.

Заметки

Функция QUEUEMARKEREVENT дает разработчикам возможность обращаться к своему коду из ячейки таблицы свойств фигуры и передавать в него данные, связанные с конкретным решением. Когда вычисляется ячейка, содержащая формулу с функцией QUEUEMARKEREVENT, приложение запускает событие маркера и передает строку события event_string всем обработчикам событий, ожидающим события типа MarkerEvent.

Дополнительные сведения о событиях маркеров см. в разделах, посвященных методу QueueMarkerEvent и событию MarkerEvent в справочнике по автоматизации Microsoft Visio.

Пример

QUEUEMARKEREVENT ("МоеПользовательскоеУведомление")

Предписывает приложению запустить событие маркера и передает строку "МоеПользовательскоеУведомление" обработчикам событий, ожидающим события типа MarkerEvent.
