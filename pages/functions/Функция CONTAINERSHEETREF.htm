Функция CONTAINERSHEETREF

Возвращает ссылку листа на указанный контейнер, который содержит фигуру.

Сведения о версии
 Добавляемая версия:  Visio 2010

Синтаксис

CONTAINERSHEETREF(index[, category])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

index

Требуется

Целое

Отсчитываемое от 1 значение индекса контейнера. Дополнительные сведения см. в разделе "Заметки".

category

Необязательно

Строка

Категория контейнера. Дополнительные сведения см. в разделе "Заметки".

Возвращаемое значение
Ссылка ShapeSheet

Заметки

Индекс контейнера вычисляется на основе порядковых номеров контейнеров по оси z, начиная с самого дальнего.

Категории — это пользовательские строки, которые можно применять для классификации фигур. Категории можно определить в ячейке User.msvShape на листе ShapeSheet для фигуры. Чтобы задать несколько категорий для фигуры, разделите их точкой с запятой.

Если фигура не является компонентом контейнера или если контейнер, соответствующий указанному номеру индекса и категории, не существует, CONTAINERSHEETREF возвращает ошибку #ССЫЛКА!.

Пример

CONTAINERSHEETREF(1)!Height

Возвращает значение в ячейке Height контейнера, находящегося на переднем плане страницы, к которой относится фигура.
