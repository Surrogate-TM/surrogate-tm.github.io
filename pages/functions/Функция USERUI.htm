Функция USERUI

Вычисляет значение одного из двух выражений в зависимости от значения параметра state.

Синтаксис

USERUI(state,defaultexpression,userexpression)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

state

Требуется

Логический

Определяет, какое выражение следует вычислить.

defaultexpression

Требуется

Строка

Выражение по умолчанию.

userexpression

Требуется

Строка

Пользовательское выражение.

Заметки

Если аргумент state равен 0, функция USERUI вычисляет значение выражения defaultexpression. Если аргумент state равен 1, вычисляется значение выражения userexpression.

Пример

USERUI(1; if(Width>6см; 6см; Width); Width*0,75)

Вычисляет значение выражения Width*0,75 и возвращает результат.
