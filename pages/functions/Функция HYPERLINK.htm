Функция HYPERLINK

Переход по указанному адресу, который может являться путем к файлу, UNC-путем или URL-адресом.

Синтаксис

HYPERLINK("address"[,"subaddress","extrainfo",window,"frame"])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

address

Требуется

Строка

Полный или относительный путь.

subaddress

Необязательно

Строка

Место в пределах расположения, заданного аргументом address, на которое указывает ссылка. Например, если аргумент address указывает на файл Microsoft Visio, параметр subaddress может представлять собой имя страницы, для файла Microsoft Office Excel он может обозначать лист или диапазон на листе, а для URL-адреса HTML -страницы — привязку.

extrainfo

Необязательно

Строка

Служит для передачи сведений, используемых для разрешения URL-адреса, например координат гиперкарты.

window

Необязательно

Логический

Определяет, должна ли гиперссылка открываться в новом окне. Значение по умолчанию — ЛОЖЬ.

frame

Необязательно

Строка

Задает имя целевого фрейма, когда приложение Visio открыто как активный документ в браузере с поддержкой ActiveX, например Microsoft Internet Explorer 3.0 или более поздней версии. Значение по умолчанию — пустая строка.

Заметки

Если для документа не существует основного пути, выполняется переход в соответствии с путем документа. Если документ не был сохранен, гиперссылка не определена.

Относительные пути определяются на основе поля База гиперссылки, определенного в диалоговом окне Visio Properties.

Для навигации по страницам документа можно использовать функцию GOTOPAGE.

Пример 1

HYPERLINK("C:\Мои документы\Документ1.vsd")

Пример 2

HYPERLINK("\\Server\Share\Документ1.vsd")

Пример 3

HYPERLINK("http://www.microsoft.com")

Пример 4

HYPERLINK("..\данные.xls";"лист1!A1")
