Функция SEGMENTCOUNT

Возвращает число сегментов линии, которые составляют путь.

Синтаксис

SEGMENTCOUNT(pathRef)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

pathRef

Требуется

Числовой

Раздел Geometry, представляющий путь, заданный ссылкой на ячейку Path (например, Geometry1. Path).

Возвращаемое значение
Числовое

Пример

Переходы по линиям не включаются в общее число возвращаемых сегментов.
