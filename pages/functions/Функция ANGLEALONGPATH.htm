Функция ANGLEALONGPATH

Возвращает угол между касательными указанной точки на пути.

Сведения о версии
 Добавляемая версия:  Visio 2010

Синтаксис

ANGLEALONGPATH(section,travel [,segment])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

section

Требуется

Строка

Раздел "Geometry", представляющий путь, который задается путем ссылки на ячейку Path (например, Geometry1.Path).

travel

Требуется

Двойное

Процент пути от начальной до конечной точки. Значение должно находиться в диапазоне от 0 до 1.

segment

Необязательно

Целое

Сегмент с индексом, отсчитываемым от 1, для которого требуется вычислить угол между касательными.

Возвращаемое значение
Двойное

Заметки

Если включить значение segment, функция ANGLEALONGPATH возвращает значение только для заданного сегмента.

Если задано значение segment, функция ANGLEALONGPATH определяет точку касания путем вычисления процента относительно значения segment с помощью значения travel.

Если section или segment не существует, возвращается ошибка #ССЫЛКА!.
