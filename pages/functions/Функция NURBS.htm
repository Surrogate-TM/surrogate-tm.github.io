Функция NURBS

Возвращает неоднородный рациональный базисный сплайн (B-сплайн). Эта функция используется в ячейке E геометрических рядов NURBSTo.

Синтаксис

NURBS(knotLast, degree, xType, yType, x1, y1, knot1, weight1, ...)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

knotLast

Требуется

Строка

Последний узел.

degree

Требуется

Числовой

Степень сплайна.

xType

Требуется

Числовой

Определяет способ интерпретации входных данных по x. Если значение xType равно 0, все входные данные по x интерпретируются как проценты от ширины. Если значение xType равно 1, все входные данные по x интерпретируются как локальные координаты.

yType

Требуется

Числовой

Определяет способ интерпретации входных данных по y. Если значение yType равно 0, все входные данные по y интерпретируются как проценты от высоты. Если значение yType равно 1, все входные данные по y интерпретируются как локальные координаты.

x1

Требуется

Строка

Координата x.

y1

Требуется

Строка

Координата y.

knot1

Требуется

Строка

Узел B-сплайна.

weight1

Требуется

Строка

Вес B-сплайна.

Заметки

Для каждого аргумента x необходимо указать аргумент y; в противном случае функция возвращает ошибку.

Необходимо указать хотя бы по одному значению аргументов x, y, knot и weight; в противном случае приложение возвращает ошибку.
