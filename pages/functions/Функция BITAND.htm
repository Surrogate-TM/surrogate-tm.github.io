Функция BITAND

Возвращает 16-разрядное двоичное число, в котором для каждого разряда задается значение 1 только в том случае, если соответствующий разряд в аргументах binarynumber1 иbinarynumber2 имеет значение 1. В противном случае для разряда задается значение 0.

Синтаксис

BITAND(binarynumber1,binarynumber2)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

binary number1

Требуется

Число

Первое 16-разрядное двоичное число.

binary number2

Требуется

Число

Второе 16-разрядное двоичное число.

Заметки
Эту функцию можно использовать для проверки и изменения свойств фигуры, которые хранятся как битовые маски (например, для изменения формата текста фигуры).

Пример

BITAND(12;6)

Возвращает 4. 12 = 0...01100. 6 = 0...00110. Следовательно, BITAND(12;6) = 0...00100.
