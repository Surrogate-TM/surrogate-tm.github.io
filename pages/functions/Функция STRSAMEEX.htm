Функция STRSAMEEX

Проверяет тождественность двух строк.

Синтаксис

STRSAMEEX ("string1", "string2", localeID, flag)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

string1

Требуется

Строка

Первая строка для сравнения.

string2

Требуется

Строка

Вторая строка для сравнения.

localeID

Требуется

Числовой

Код языка.

flag

Требуется

Числовой

Разряд, определяющий тип сравнения.

Возвращаемое значение
Логическое

Заметки

Функция STRSAMEEX возвращает значение ИСТИНА, если строки тождественны, и ЛОЖЬ, если нет. Эта функция используется для сравнения многобайтовых строк и для сравнения строк с учетом правил использования регистра для определенного языкового стандарта.

В функции STRSAMEEX можно использовать любое сочетание описанных ниже флагов.

Флаг

Описание

1

Не учитывать регистр.

2

Не учитывать несамостоятельные знаки.

4

Не учитывать символы.

4096

Рассматривать знаки препинания как символы.

65536

Не различать символы хираганы и катаканы.

131072

Не различать однобайтовые символы и идентичные им многобайтовые символы.
