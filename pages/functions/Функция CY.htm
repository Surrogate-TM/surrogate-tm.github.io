Функция CY

Возвращает денежное значение.

Синтаксис

CY(value, cyID)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

value

Необязательно

Число или строка

Число или строка, включающая форматирование денежной единицы. Если форматирование не указано, используются значения стиля денежной единицы, заданные в системных параметрах в окне "Язык и региональные стандарты".

cyID

Необязательно

Число

Числовой идентификатор денежной единицы или строка из трех символов в кавычках (аббревиатура ISO 4217).

Заметки

Чтобы указать другую денежную единицы, необходимо задать допустимое значение cyID. Список значений см. в разделе Денежные константы.

Если значение value несовместимо с назначенным типом денежной единицы или задан недопустимый аргумент, например нечисловое значение, возвращается ошибка #ЗНАЧ!. Если value больше 922 337 203 685 477,5807 или меньше -922 337 203 685 477 5808, возвращается ошибка #ЗНАЧ!.

Для более точного вычисления очень больших денежных значений, которые включают дроби (таких как 3,6 триллиона), используйте строковые аргументы для параметра value.

Недопустимое значение cyID возвращает ошибку.

Пример 1

Если в параметрах "Язык и региональные стандарты" пользователя заданы доллары США:

CY(999998,993)

Возвращается значение $999 998,99

Пример 2

CY(12345678,993; "USD")

Возвращается значение $12 345 678,99

Пример 3

CY(12345678,993; "DEM")

Возвращается значение 12 345 678,99 DEM

Пример 4

CY(12345678,7832; "XXX")

Возвращается значение 12 345 678,78
