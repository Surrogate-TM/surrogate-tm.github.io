Функция CALLOUTTARGETREF

Возвращает ссылку листа на конечную фигуру, с которой связана данная фигура выноски.

Сведения о версии
 Добавляемая версия:  Visio 2010

Синтаксис

CALLOUTTARGETREF()!

Возвращаемое значение
Ссылка ShapeSheet

Заметки
Если фигура не является фигурой выноски или не связана с конечной фигурой, функция CALLOUTTARGETREF возвращает значение #ССЫЛКА.

Пример

CALLOUTTARGETREF()!Height

Возвращает значение в ячейке Height фигуры, связанной с выноской.
