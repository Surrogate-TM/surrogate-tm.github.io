Функция CELLISTHEMED

Позволяет указать для фигуры одно поведение, когда к ней применяется тема, и другое, когда тема не применяется.

Синтаксис

CELLISTHEMED()

Заметки

Функция CELLISTHEMED позволяет использовать нужные элементы форматирования темы для фигуры, переопределяя при необходимости форматирование темы для других элементов форматированием, примененным вручную. Если после этого повторно применить тему, все форматирование, примененное вручную, будет переопределено, а для фигуры будет использоваться все форматирование темы.

Пример

Visual Basic для приложений

Shape.FillForegnd = THEMEGUARD(IF(CELLISTHEMED(TRUE),THEME("AccentColor2"),SETATREFEXPR()))

Если CELLISTHEMED имеет значение TRUE, в качестве цвета переднего плана заливки для фигуры будет использоваться цвет "Акцент 2". Если затем применить цвет заливки вручную, для CELLISTHEMED устанавливается значение FALSE, а заданный вручную цвет сохраняется в SETATREFEXPR.
