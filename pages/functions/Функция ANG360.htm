Функция ANG360

Нормализует диапазон угла до 0 <= результат < 2ПИ радиан (0 <= результат < 360 градусов).

Синтаксис

ANG360(angle)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

angle

Требуется

Число

Нормализуемый угол.

Заметки

Если угол не задан в единицах измерения угла, считается, что он указан в радианах. Если не удается преобразовать угол в значение, возвращается ошибка #ЗНАЧ!.

Пример 1

ANG360(395 град.)

Возвращает 35 градусов

Пример 2

ANG360(-9,8 рад)

Возвращает 2,7664 радианы

Пример 3

ANG360(45)

Возвращает 58,31 градусов (1,0177 радианы)
