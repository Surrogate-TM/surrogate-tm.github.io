Функция OPENFILE

Открывает документ Microsoft Visio, если он еще не открыт, и активирует окно документа.

Синтаксис

OPENFILE("filename")

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

filename

Требуется

Строка

Имя открываемого файла.

Заметки

Множественные вызовы функции OPENFILE помещаются в очередь и выполняются в порядке получения. Если текущий документ Visio активирован для визуального редактирования (по месту), запускается новый экземпляр программы с запрошенным файлом.

Эта функция всегда возвращает значение ЛОЖЬ.

В более ранних версиях Visio эта функция называлась _OPENFILE. В приложении Visio 4.0 и более поздних версий допустим любой вариант.

Пример

OPENFILE("basic.vss")

Открывает указанный файл набора элементов Basic.vss в новом окне или активирует окно, если соответствующий набор элементов уже открыт.
