Функция SINH

Возвращает значение гиперболического синуса угла.

Синтаксис

SINH(angle)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

angle

Требуется

Числовой

Угол, гиперболический синус которого требуется получить.

Возвращаемое значение
Число

Заметки

В функции SINH используются радианы, если в параметре angle не указана другая единица измерения.

Пример

SINH(45 градусов)

Возвращает 0,8687.
