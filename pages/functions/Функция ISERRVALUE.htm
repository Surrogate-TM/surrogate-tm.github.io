Функция ISERRVALUE

Возвращает значение ИСТИНА, если значение cellreference является ошибкой типа #ЗНАЧ!, а аргумент в формуле имеет недопустимый тип. Функция ISERRVALUE используется в логических выражениях, ссылающихся на другие ячейки.

Синтаксис

ISERRVALUE(cellreference)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

cellreference

Требуется

Строка

Ссылка на ячейку.

Заметки

Вспомогательные ячейки с A по D не возвращают ошибку #ЗНАЧ!, поскольку формула может содержать в одной строке как числа, так и буквы. Ячейки X и Y должны содержать только числа.

Пример 1

Ячейка

Формула

Возвращаемое значение

Scratch.X1

="Дом"

#ЗНАЧ!

Scratch.A1

= If (ISERRVALUE(Scratch.X1);2;Scratch.X1)

2

Возвращает 2, поскольку возвращаемое значение является ошибкой #ЗНАЧ!, а выражение предписывает Microsoft Visio вернуть вместо ошибки значение 2.

Пример 2

Ячейка

Формула

Возвращаемое значение

Scratch.A1

="5 + 7"

5 + 7

Scratch.B1

=If (ISERRVALUE(Scratch.A1);2;Scratch.A1)

5 + 7

Возвращает 12, поскольку возвращаемое значение не является ошибкой #ЗНАЧ!, а выражение предписывает Visio вернуть значение исходной ячейки.
