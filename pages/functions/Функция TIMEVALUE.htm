Функция TIMEVALUE

Возвращает значение времени, представленное аргументом second или expression, с учетом системных параметров языка и региональных стандартов.

Синтаксис

TIMEVALUE("datetime"|expression[,lcid])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

datetime

Требуется

Строка

Любая строка с датой и временем в общепринятом формате или ссылка на ячейку, содержащую дату и время.

expression

Требуется

Изменяется

Любое выражение, возвращающее дату и время.

lcid

Необязательно

Число

Код языка для использования при анализе значения даты и времени в формате другого региона. Код языка — это число, описанное в системных заголовочных файлах.

Заметки

Компонент даты в аргументах datetime и expression отбрасывается.

Если значение datetime отсутствует или не может быть преобразовано в допустимый результат, функция возвращает ошибку #ЗНАЧ!.

Функция TIMEVALUE также принимает в качестве параметра expression отдельное числовое значение, десятичная часть которого представляет долю суток, истекшую с полуночи.

Пример 1

TIMEVALUE("6:00")

Возвращает значение 06:00:00.

Пример 2

TIMEVALUE("14:30")+4 час+30 мин

Возвращает значение 19:00:00.

Пример 3

TIMEVALUE("1 июля 1997 г., 11:00")

Возвращает значение 11:00:00.

Пример 4

TIMEVALUE(0,6337)

Возвращает значение 15:12:32.

Пример 5

TIMEVALUE("7:89")

Возвращает ошибку #ЗНАЧ!.
