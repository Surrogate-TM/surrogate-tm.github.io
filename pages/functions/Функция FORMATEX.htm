Функция FORMATEX

Возвращает результат выражения expression, рассчитанного в srcUnit, в виде строки в формате, соответствующем формату format в dstUnit.

Синтаксис

FORMATEX(expression,"format",[srcUnit],[dstUnit],[langID][,calID])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

expression

Требуется

Строка

Сочетание констант, операторов, функций и ссылок на ячейки ShapeSheet, результатом которых является значение.

format

Требуется

Строка

Представление формата, используемое для форматирования строки.

srcUnit

Необязательно

Строка

Единицы измерения, используемые для вычисления выражения expression (дюйм, см и т. д.).

dstUnit

Необязательно

Строка

Единицы измерения, использующие результат выражения expression (дюйм, см и т. д.).

langID

Необязательно

Число

Язык, используемый для определения представлений формата даты и времени системы Microsoft Office.

calID

Необязательно

Число

Календарь, используемый для определения представлений формата даты и времени системы Microsoft Office.

Возвращаемое значение
Строка

Заметки

Поведение возвращаемой строки определяется типом выражения и типом, заданным в представлении формата. Значение format должно соответствовать типу выражения.

Аргумент srcUnit используется для обработки нетипизированных результатов выражения, таких как 3 + 4. Если результат выражения expression имеет явный тип (например, "3 см + 8 см"), значение srcUnit игнорируется.

Аргумент dstUnit позволяет задать единицы измерения для отформатированного результата. Если аргумент dstUnit не задан, используются единицы результата выражения.

Если тип результата expression не соответствует типу format, в format есть синтаксические ошибки или единицы измерения, переданные в качестве аргументов srcUnit или dstUnit, не распознаются, возвращается ошибка.

Пример

FORMATEX(5,5; "0,00 u"; "см", "фт")

Возвращает 0,18 фута.
