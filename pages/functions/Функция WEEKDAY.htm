Функция WEEKDAY

Возвращает целое число от 1 до 7, соответствующее дню недели в аргументе datetime или expression.

Синтаксис

WEEKDAY("datetime"|expression[,lcid])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

datetime

Требуется

Строка

Любая строка с датой и временем в общепринятом формате или ссылка на ячейку, содержащую дату и время.

expression

Требуется

Изменяется

Любое выражение, возвращающее дату и время.

lcid

Необязательно

Числовой

Код языка для использования при анализе значения даты и времени в формате другого региона. Код языка — это число, описанное в системных заголовочных файлах.

Возвращаемое значение
Целое число

Заметки

Компонент времени в аргументах datetime и expression отбрасывается.

Результат соответствует дням недели с понедельника (1) по воскресенье (7). Округление не выполняется. Если аргумент datetime отсутствует или не может быть интерпретирован как допустимое значение, функция возвращает ошибку #ЗНАЧ!.

Функция WEEKDAY также принимает в качестве параметра expression отдельное числовое значение, целая часть которого представляет количество дней, прошедших с 30 декабря 1899 года.

Пример 1

WEEKDAY("30 мая 1999 г.")

Возвращает 7.

Пример 2

WEEKDAY(DATEVALUE("30 мая 1999 г.")+2 дн)

Возвращает 2.

Пример 3

WEEKDAY(35880,6337)

Возвращает 4.
