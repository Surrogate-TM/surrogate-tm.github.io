Функция FONT Появилась в версии 2013

Возвращает целочисленное значение уникального идентификатора для шрифта, заданного по имени.

Синтаксис

FONT("font_name_string")

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

font_name_string

Требуется

Текстовый

Название шрифта.

Возвращаемое значение
Числовое

Примечания

Если строка, указанная как имя шрифта не совпадает с известным шрифтом, эта функция возвращает ошибку #VALUE!.

Пример

FONT("Calibri")

Возвращает целочисленное значение (4), представляющее уникальный идентификатор шрифта "Calibri".
