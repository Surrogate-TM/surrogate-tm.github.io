Функция PATHSEGMENT

Возвращает значение ИСТИНА, если указанная строка найдена в списке категорий фигуры.

Синтаксис

PATHSEGMENT(category)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

logicalexpression

Требуется

Строка

Выражение, значение которого требуется определить.

valueiftrue

Требуется

Изменяется

Возвращаемое выражение, если параметр logicalexpression имеет значение ИСТИНА.

valueiffalse

Требуется

Изменяется

Возвращаемое выражение, если параметр logicalexpression имеет значение ЛОЖЬ.

Возвращаемое значение
Логическое

Примечания

Категории — это определяемые пользователем строковые переменные, которые можно использовать для определения категорий фигур.
Можно определить категории в ячейке User.msvShapeCategories таблицы свойств фигуры. Можно задать несколько категорий для фигуры, разделив категории с запятой.
