Функция TIME

Возвращает время, представленное значениями hour, minute и second.

Синтаксис

TIME(hour,minute,second)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

hour

Требуется

Числовой

Компонент часов.

minute

Требуется

Числовой

Компонент минут.

second

Требуется

Числовой

Компонент секунд.

Возвращаемое значение
Числовое

Пример 1

TIME(15;30;30)

Возвращает значение времени 15:30:30.

Пример 2

FORMAT(TIME(15;30;30);"ЧЧ:мм")

Возвращает значение времени 15:30.

Пример 3

TIME(15;30;30) + 8 час

Возвращает значение времени 23:30:30.
