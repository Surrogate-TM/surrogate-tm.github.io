Функция GRAVITY

Вычисляет угол поворота блока текста для указанного поворота фигуры, чтобы избежать переворачивания текста.

Синтаксис

GRAVITY(angle,[limit1],[limit2])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

angle

Требуется

Строка

Угол фигуры.

limit1

Необязательно

Строка

Первое ограничение вращения. Значение по умолчанию — 90 градусов.

limit2

Необязательно

Строка

Второе ограничение вращения. Значение по умолчанию — 270 градусов.

Возвращаемое значение
Строка

Заметки

Функция GRAVITY обычно используется в ячейке TxtAngle.

Возвращается значение 180 градусов, если angle находится между значениями, заданными аргументами limit1 и limit2; в противном случае возвращается значение 0 градусов.

Все аргументы автоматически нормализуются функцией в пределах от 0 до 360 градусов. Если в аргументе не заданы единицы измерения, используются радианы.

Пример 1

GRAVITY(Angle)

Возвращается значение 180 градусов, если angle находится в пределах от 90 до 270 градусов; в противном случае возвращается значение 0 градусов.

Пример 2

GRAVITY(2)

Возвращает 180, так как 2 радианы находятся в пределах от 90 до 270 градусов.

Пример 3

GRAVITY(100 град, 110 град, 290 град)

Возвращает 0 градусов.

Пример 4

GRAVITY(100 град, 290 град, 110 град)

Возвращает 0 градусов.
