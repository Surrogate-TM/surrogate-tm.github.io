Функция DEFAULTEVENT

Выполняет событие по умолчанию, связанное с объектом.

Синтаксис

DEFAULTEVENT( )

Заметки

В следующей таблице указаны события по умолчанию, связанные с некоторыми объектами.

Объект

Событие по умолчанию

Фигура

Изменение текста.

Группа

Нет действий.

OLE

Выполнение основной команды.

Другой

Нет действий.

В предыдущих версиях Visio эта функция называлась _DEFAULTEVENT. Событием по умолчанию для группы является открытие окна группы.
