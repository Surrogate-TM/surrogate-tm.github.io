Функция DEPENDSON

Создает зависимость для ссылки на ячейку.

Синтаксис

DEPENDSON(cellref[,cellref2,...])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

cellref

Требуется

Строка

Первая ссылка на ячейку.

cellref2

Необязательно

Строка

Вторая ссылка на ячейку.

Заметки

Эта функция всегда возвращает значение FALSE. Она не применяется при использовании в строке Event или в ячейке Action.

Пример

OPENTEXTWIN() + DEPENDSON(PinX,PinY)

Открывает блок текста для фигуры при изменении ее ячейки PinX или PinY.
