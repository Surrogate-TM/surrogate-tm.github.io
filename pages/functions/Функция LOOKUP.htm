Функция LOOKUP

Возвращает индекс, отсчитываемый от нуля, который обозначает расположение подстроки key в списке list, или значение -1, если целевая строка содержит разделитель delimiter.

Синтаксис

LOOKUP("key","list"[,"delimiter"])

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

key

Требуется

Строка

Строка, которую требуется найти.

list

Требуется

Строка

Список, в котором выполняется поиск.

delimiter

Необязательно

Строка

Строка, используемая в списке list в качестве разделителя. Эта строка может состоять из одного или нескольких знаков и содержать многобайтовые символы. Значение по умолчанию — точка с запятой.

Возвращаемое значение
Числовое

Заметки

Функция LOOKUP выполняет поиск без учета регистра. Если список начинается с разделителя или заканчивается им, считается, что перед списком или после него находится строка нулевой длины. Между последовательными разделителями также располагается строка нулевой длины.

Все аргументы функции должны быть строками или выражениями, которые можно преобразовать в строки. В противном случае аргумент заменяется пустой строкой.

Пример 1

LOOKUP("крыса";"кот;крыса;;коза")

Возвращает 1.

Пример 2

LOOKUP("";";кот;кобра;;коза")

Возвращает 0.

Пример 3

LOOKUP("т";"кот;крыса;;коза";"о")

Возвращает 3.
