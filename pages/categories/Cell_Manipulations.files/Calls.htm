Управление вызовами

Примечания

Эта таблица содержит только функции, которые были включены в таблицу с аналогичной категорией в книге MS Visio 2003 Developer’s Survival Pack от Graham Wideman (см. главу 37).

Информация о версии, в которой была добавлена функция взята из статьи Visio ShapeSheet Functions by Version от John's Goldsmith

Название функции

Описание функции

CALLTHIS

Добавлена в версии: до 2003

Вызывает процедуру в проекте Microsoft Visual Basic для приложений (VBA).

Синтаксис

CALLTHIS("procedure",["project"],[arg1,arg2,...])

Подробнее о функции CALLTHIS

DOCMD

Добавлена в версии: до 2003

Выполняет указанную команду.

Синтаксис

DOCMD (commandID)

Подробнее о функции DOCMD

DOOLEVERB

Добавлена в версии: до 2003

Выполняет команду для объекта OLE.

Синтаксис

DOOLEVERB("verb")

Подробнее о функции DOOLEVERB

RUNADDON

Добавлена в версии: до 2003

Выполняет дополнительный компонент или макрос в проекте Microsoft Visual Basic для приложений (VBA).

Синтаксис

RUNADDON("string")

Подробнее о функции RUNADDON

RUNADDONWARGS

Добавлена в версии: до 2003

Запускает дополнительный компонент с именем, заданным параметром string, и передает в программу аргументы командной строки, заданные строкой arguments.

Синтаксис

RUNADDONWARGS("string","arguments")

Подробнее о функции RUNADDONWARGS
