Служебные ошибки — категория функций таблицы свойств (ShapeSheet)

Примечания

Эта таблица содержит функции, которые были включены в таблицу с аналогичной категорией в книге MS Visio 2003 Developer’s Survival Pack от Graham Wideman (см. главу 37), а также функции добавленные в следующих версиях.

Информация о версии, в которой была добавлена функция взята из статьи Visio ShapeSheet Functions by Version от John's Goldsmith

Название функции

Описание функции

IFERROR

Добавлена в версии: 2010

Возвращает вычисленный результат первичного выражения, если он не выдает ошибку. В противном случае возвращает вычисленный результат альтернативного выражения.

Синтаксис

IFERROR(primary expression, alternate expression)

Подробнее о функции IFERROR

ISERR

Добавлена в версии: до 2003

Возвращает значение ИСТИНА, если аргумент cellreference является ошибкой любого типа, кроме #Н/Д!; в противном случае возвращает значение ЛОЖЬ. Функция ISERR используется в формулах, ссылающихся на другие ячейки.

Синтаксис

ISERR(cellreference)

Подробнее о функции ISERR

ISERRNA

Добавлена в версии: до 2003

Возвращает значение ИСТИНА, если аргумент cellreference является ошибкой типа #Н/Д! (недоступно); в противном случае возвращает значение ЛОЖЬ. Функция ISERRNA используется в формулах, ссылающихся на другие ячейки.

Синтаксис

ISERRNA(cellreference)

Подробнее о функции ISERRNA

ISERROR

Добавлена в версии: до 2003

Возвращает значение ИСТИНА, если аргумент cellreference является ошибкой любого типа; в противном случае возвращает значение ЛОЖЬ. Функция ISERROR используется в формулах, ссылающихся на другие ячейки.

Подробнее о функции ISERROR

ISERRVALUE

Добавлена в версии: до 2003

Возвращает значение ИСТИНА, если значение cellreference является ошибкой типа #ЗНАЧ!, а аргумент в формуле имеет недопустимый тип. Функция ISERRVALUE используется в логических выражениях, ссылающихся на другие ячейки.

Синтаксис
