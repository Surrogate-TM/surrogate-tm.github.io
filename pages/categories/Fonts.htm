Свойства шрифтов — категория функций таблицы свойств (ShapeSheet)

Примечания

Информация о версии, в которой была добавлена функция взята из статьи Visio ShapeSheet Functions by Version от John's Goldsmith

Название функции

Описание функции

FONT

Добавлена в версии: 2013

Возвращает целочисленное значение уникального идентификатора для шрифта, заданного по имени.

Синтаксис

FONT("font_name_string")

Подробнее о функции FONT

FONTTOID

Добавлена в версии: 2010

Возвращает идентификатор (ID) указанного шрифта.

Синтаксис

FONT("font_name")
