Управление событиями и действиями — категория функций таблицы свойств (ShapeSheet)

Примечания

Эта таблица содержит только функции, которые были включены в таблицу с аналогичной категорией в книге MS Visio 2003 Developer’s Survival Pack от Graham Wideman (см. главу 37).

Информация о версии, в которой была добавлена функция взята из статьи Visio ShapeSheet Functions by Version от John's Goldsmith

Название функции

Описание функции

DEFAULTEVENT

Добавлена в версии: до 2003

Выполняет событие по умолчанию, связанное с объектом.

Синтаксис

DEFAULTEVENT( )

Подробнее о функции DEFAULTEVENT

GOTOPAGE

Добавлена в версии: до 2003

Отображает страницу с именем pagename в активном окне..

Синтаксис

GOTOPAGE("pagename")

Подробнее о функции GOTOPAGE

HELP

Добавлена в версии: до 2003

Открывает файл справки HTML с указанным ключевым словом (keyword) в поле Поиск.

Синтаксис

HELP("filename.chm!keyword")

Подробнее о функции HELP

HYPERLINK

Добавлена в версии: до 2003

Переход по указанному адресу, который может являться путем к файлу, UNC-путем или URL-адресом.

Синтаксис

HYPERLINK("address"[,"subaddress","extrainfo",window,"frame"])

Подробнее о функции HYPERLINK

OPENFILE

Добавлена в версии: до 2003

Открывает документ Microsoft Visio, если он еще не открыт, и активирует окно документа.

Синтаксис

OPENFILE("filename")

Подробнее о функции OPENFILE

OPENGROUPWIN

Добавлена в версии: до 2003

Открывает группу в окне группы. Если объект не является группой, никакие действия не выполняются.

Синтаксис

OPENGROUPWIN( )

Подробнее о функции OPENGROUPWIN

OPENSHEETWIN

Добавлена в версии: до 2003

Открывает окно таблицы свойств фигуры в новом окне.

Синтаксис

OPENSHEETWIN( )

Подробнее о функции OPENSHEETWIN

OPENTEXTWIN

Добавлена в версии: до 2003

Открывает блок текста фигуры для редактирования текста.

Синтаксис

OPENTEXTWIN( )

Подробнее о функции OPENTEXTWIN

PLAYSOUND

Добавлена в версии: до 2003

Воспроизводит звуковой файл или системный звук.

Синтаксис

PLAYSOUND("filename"|"alias",isAlias,beep,synch)

Подробнее о функции PLAYSOUND

Синтаксис
