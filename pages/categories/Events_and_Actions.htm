HELP

Открывает файл справки HTML с указанным ключевым словом (keyword) в поле Поиск.

Синтаксис

Подробнее о функции HELP

HYPERLINK

Переход по указанному адресу, который может являться путем к файлу, UNC-путем или URL-адресом.

Синтаксис

HYPERLINK("address"[,"subaddress","extrainfo",window,"frame"])

Подробнее о функции HYPERLINK
