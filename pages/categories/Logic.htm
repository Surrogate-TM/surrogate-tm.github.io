Логические функции — категория функций таблицы свойств (ShapeSheet)

Примечания

Эта таблица содержит только функции, которые были включены в таблицу с аналогичной категорией в книге MS Visio 2003 Developer’s Survival Pack от Graham Wideman (см. главу 37).

Описание функции

AND

Добавлена в версии: до 2003

Возвращает значение ИСТИНА (1), если все предоставленные логические выражения имеют значение ИСТИНА. Если хотя бы одно из логических выражений имеет значение ЛОЖЬ или 0, функция И возвращает значение ЛОЖЬ (0).

Синтаксис

AND(logical expression1,logical expression2,...,logical expressionN)

Подробнее о функции AND

BITAND

Добавлена в версии: до 2003

Возвращает 16-разрядное двоичное число, в котором для каждого разряда задается значение 1 только в том случае, если соответствующий разряд в аргументах binarynumber1 иbinarynumber2 имеет значение 1. В противном случае для разряда задается значение 0.

Синтаксис

BITAND(binarynumber1,binarynumber2)

Подробнее о функции BITAND

BITNOT

Добавлена в версии: до 2003

Возвращает 16-разрядное двоичное число, в котором для каждого разряда задается значение 1 только в случае, если соответствующий разряд в аргументе binary number имеет значение 0. В противном случае для разряда задается значение 0.

Синтаксис

BITNOT(binary number)

Подробнее о функции BITNOT

BITOR

Добавлена в версии: до 2003

Возвращает 16-разрядное двоичное число, в котором для каждого разряда задается значение 1, если соответствующий разряд в любом из аргументов binary number1 или binary number2 имеет значение 1. Значение 0 задается только в случае, если соответствующий разряд имеет значение 0 в обоих аргументах binary number1 и binary number2.

Синтаксис

BITOR(binary number1,binary number2)

Подробнее о функции BITOR

BITXOR

Добавлена в версии: до 2003

Возвращает 16-разрядное двоичное число, в котором для каждого разряда задается значение 1, если соответствующий разряд имеет значение 1 только в одном из аргументов binary number1 или binary number2. В противном случае для разряда задается значение 0.

Синтаксис

BITXOR(binary number1,binary number2)

Подробнее о функции BITXOR

NOT

Добавлена в версии: до 2003

Возвращает значение ИСТИНА (1), если аргумент logicalexpression имеет значение ЛОЖЬ. В противном случае возвращает значение ЛОЖЬ (0).

Синтаксис

NOT(logicalexpression)

Подробнее о функции NOT

OR

Добавлена в версии: до 2003

Возвращает значение ИСТИНА (1), если хотя бы одно из логических выражений, переданных с помощью аргументов, имеет значение ИСТИНА.

Синтаксис

OR(logicalexpression1,logicalexpression2,...,logicalexpressionN)
